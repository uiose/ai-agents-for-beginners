>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git

## 🌱 شروعات

یہ کورس AI ایجنٹس بنانے کے بنیادی اصولوں کا احاطہ کرنے والے اسباق پر مشتمل ہے۔ ہر سبق اپنا مخصوص موضوع کور کرتا ہے، لہٰذا جہاں چاہیں وہیں سے شروع کریں!

اس کورس کے لیے متعدد زبانوں کی حمایت موجود ہے۔ ہماری [دستیاب زبانیں یہاں](../..) دیکھیں۔ 

اگر یہ آپ کا پہلی بار Generative AI ماڈلز کے ساتھ بلڈنگ ہے، تو ہمارا [جنریٹو AI برائے مبتدی](https://aka.ms/genai-beginners) کورس دیکھیں، جس میں GenAI کے ساتھ بنڈنگ پر 21 اسباق شامل ہیں۔

یہ ریپو [اسے اسٹار (🌟) دینا نہ بھولیں](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) اور کوڈ چلانے کے لیے [اس ریپو کو فورک کریں](https://github.com/microsoft/ai-agents-for-beginners/fork)۔

### دوسرے سیکھنے والوں سے ملیں، اپنے سوالات کے جواب حاصل کریں

اگر آپ پھنس جائیں یا AI ایجنٹس بنانے کے بارے میں کوئی سوال ہو تو ہمارے مختص شدہ ڈسکارڈ چینل میں شامل ہوں: [Microsoft Foundry ڈسکارڈ](https://aka.ms/ai-agents/discord)۔

### آپ کو کیا چاہیے

ہر سبق میں کوڈ مثالیں شامل ہیں، جو code_samples فولڈر میں مل سکتی ہیں۔ آپ اپنی کاپی بنانے کے لیے [اس ریپو کو فورک](https://github.com/microsoft/ai-agents-for-beginners/fork) کر سکتے ہیں۔  

ان مشقوں میں موجود کوڈ مثالیں Microsoft Agent Framework کے ساتھ Azure AI Foundry Agent Service V2 استعمال کرتی ہیں:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure اکاؤنٹ درکار ہے

یہ کورس Microsoft کے مندرجہ ذیل AI ایجنٹ فریم ورکس اور سروسز استعمال کرتا ہے:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


مزید معلومات کے لیے کہ اس کورس کے کوڈ کو کیسے چلایا جائے، [Course Setup](./00-course-setup/README.md) دیکھیں۔
