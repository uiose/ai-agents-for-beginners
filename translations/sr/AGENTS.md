# AGENTS.md

## Преглед Пројекта

Овај репозиторијум садржи "AI Агенте за Почетнике" – свеобухватан образовни курс који учи све што је потребно за изградњу AI агената. Курс се састоји од 15+ поука који покривају основе, дизајн обрасце, оквире и продукционо имплементирање AI агената.

**Кључне Технологије:**
- Python 3.12+
- Jupyter Notebooks за интерактивно учење
- AI Оквири: Microsoft Agent Framework (MAF)
- Azure AI Услуге: Microsoft Foundry, Azure AI Foundry Agent Service V2

**Архитектура:**
- Структура заснована на лекцијама (директоријуми 00-15+)
- Свакa лекција садржи: документацију у README-у, примере кода (Jupyter нотебоок-ове) и слике
- Подршка за више језика преко аутоматизованог система превођења
- Један Python нотебоок по лекцији са Microsoft Agent Framework-ом

## Команде за Подешавање

### Предуслови
- Python 3.12 или новији
- Azure претплата (за Azure AI Foundry)
- Azure CLI инсталиран и аутентификован (`az login`)

### Почетно Подешавање

1. **Клонирати или форкнути репозиторијум:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # ИЛИ
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Креирати и активирати Python виртуелно окружење:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # На Виндоуз-у: venv\Scripts\activate
   ```

3. **Инсталирати зависности:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Подесити системске променљиве окружења:**
   ```bash
   cp .env.example .env
   # Уредите .env са вашим API кључевима и крајњим тачкама
   ```

### Потребне Променљиве Окружења

За **Azure AI Foundry** (потребно):
- `AZURE_AI_PROJECT_ENDPOINT` - крајња тачка Azure AI Foundry пројекта
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - име моделске имплементације (нпр. gpt-4o)

За **Azure AI Search** (лекција 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - крајња тачка Azure AI Search
- `AZURE_SEARCH_API_KEY` - API кључ за Azure AI Search

Аутентификација: Покрените `az login` пре рада у нотебоок-овима (користи `AzureCliCredential`).

## Додатне Напомене

### Чести Проблеми

1. **Разлика у верзији Python-а:**
   - Осигурајте да користите Python 3.12+
   - Неке библиотеке не раде са старијим верзијама
   - Користите `python3 -m venv` за експлицитно одређивање Python-а

2. **Променљиве окружења:**
   - Увек направите `.env` из `.env.example`
   - Немојте комитовати `.env` фајл (на списку је .gitignore)
   - GitHub токен треба одговарајуће дозволе

3. **Конфликти пакета:**
   - Користите чисто виртуелно окружење
   - Инсталирајте преко `requirements.txt` уместо појединачних пакета
   - Неке нотебоок-ове могу захтевати додатне пакете наведене у markdown ћелијама

4. **Azure услуге:**
   - Azure AI услуге захтевају активну претплату
   - Неке функције су регионално ограничене
   - Бесплатни слој има ограничења за GitHub Моделе
