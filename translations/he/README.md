> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> זה נותן לכם את כל מה שצריך כדי להשלים את הקורס עם הורדה מהירה בהרבה.

## 🌱 התחלה

בקורס זה יש שיעורים המכסים את יסודות בניית סוכני בינה מלאכותית. כל שיעור עוסק בנושא משלו — התחילו מאיפה שתרצו!

יש תמיכה בריבוי שפות לקורס זה. עברו ל[השפות הזמינות כאן](../..). 

אם זו הפעם הראשונה שלכם בבנייה עם מודלים של בינה גנרטיבית, בדקו את הקורס שלנו [בינה גנרטיבית למתחילים](https://aka.ms/genai-beginners), שמכיל 21 שיעורים על בנייה עם GenAI.

אל תשכחו [להעניק כוכב (🌟) למאגר זה](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ו[לשכפל (fork) את המאגר](https://github.com/microsoft/ai-agents-for-beginners/fork) כדי להריץ את הקוד.

### הכירו לומדים אחרים וקבלו תשובות לשאלותיכם

אם תתקעו או יהיו לכם שאלות לגבי בניית סוכני בינה מלאכותית, הצטרפו לערוץ ה-Discord הייעודי שלנו ב-[Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### מה תצטרכו

כל שיעור בקורס זה כולל דוגמאות קוד, שניתן למצוא בתיקיית code_samples. תוכלו [לשכפל את המאגר](https://github.com/microsoft/ai-agents-for-beginners/fork) כדי ליצור עותק משלכם.  

דוגמאות הקוד בתרגילים אלה משתמשות ב-Microsoft Agent Framework יחד עם Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - נדרש חשבון Azure

בקורס זה נעשה שימוש במסגרת השירותים והפלטפורמות הבאות של Microsoft לסוכני AI:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


למידע נוסף על הרצת הקוד לקורס זה, עברו ל-[Course Setup](./00-course-setup/README.md).
