> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git

## 🌱 Kuanzia

Kozi hii ina masomo yanayofunika misingi ya ujenzi wa Wakala wa AI. Kila somo lina mada yake mwenyewe hivyo anza mahali popote upendalo!

Kuna msaada wa lugha nyingi kwa kozi hii. Nenda kwa [lugha zinazopatikana hapa](../..).

Kama huu ni mara yako ya kwanza kujenga kwa kutumia mifano ya Generative AI, angalia kozi yetu ya [Generative AI kwa Waanzilishi](https://aka.ms/genai-beginners), ambayo ina mafundisho 21 kuhusu ujenzi na GenAI.

Usisahau [kutoa nyota (🌟) kwa repo hii](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) na [kufanya fork ya repo hii](https://github.com/microsoft/ai-agents-for-beginners/fork) ili kuendesha msimbo.

### Kutana na Wanafunzi Wengine, Pata Majibu ya Maswali Yako

Kama unakumbwa na matatizo au una maswali yoyote kuhusu ujenzi wa Wakala wa AI, jiunge na Kituo chetu maalum cha Discord katika [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Unachohitaji

Kila somo katika kozi hii lina mfano wa msimbo, unaweza kupatikana kwenye folda ya code_samples. Unaweza [kufanya fork ya repo hii](https://github.com/microsoft/ai-agents-for-beginners/fork) kuunda nakala yako mwenyewe.

Mifano ya msimbo katika mazoezi haya inatumia Microsoft Agent Framework na Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Akaunti ya Azure Inahitajika

Kozi hii inatumia mifumo na huduma za AI Agent zifuatazo kutoka Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Huduma ya Azure AI Foundry Agent V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Kwa maelezo zaidi kuhusu kuendesha msimbo wa kozi hii, nenda kwenye [Course Setup](./00-course-setup/README.md).
