>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Tai suteikia jums viską, ko reikia kursui atlikti, ir leidžia daug greičiau atsisiųsti.

## 🌱 Pradžia

Šiame kurse yra pamokos apie dirbtinio intelekto agentų kūrimo pagrindus. Kiekviena pamoka nagrinėja savo temą, tad pradėkite ten, kur norite!

Šiam kursui yra daugiakalbė palaikymas. Peržiūrėkite mūsų [pasiekiamas kalbas čia](../..). 

Jei tai pirmas kartas, kai kuriate su generatyvios DI modeliais, peržiūrėkite mūsų [Generatyvaus DI pradžiamokslį](https://aka.ms/genai-beginners) kursą, kuriame yra 21 pamoka apie kūrimą su GenDI.

Nepamirškite [užžymėti (🌟) šią saugyklą](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ir [šakoti šią saugyklą](https://github.com/microsoft/ai-agents-for-beginners/fork), kad galėtumėte paleisti kodą.

### Susipažinkite su kitais besimokančiais, gaukite atsakymus į savo klausimus

Jei užstringate ar turite klausimų apie dirbtinio intelekto agentų kūrimą, prisijunkite prie mūsų skirto Discord kanalo [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Ko jums reikia

Kiekviena pamoka šiame kurse apima kodo pavyzdžius, kuriuos rasite code_samples kataloge. Galite [šakoti šią saugyklą](https://github.com/microsoft/ai-agents-for-beginners/fork), kad sukurtumėte savo kopiją.  

Šių užduočių kodo pavyzdžiai naudoja Microsoft Agent Framework su Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - reikalinga Azure paskyra

Šiame kurse naudojami šie Microsoft DI agentų karkasai ir paslaugos:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Daugiau informacijos apie kodo paleidimą šiam kursui rasite [Kurso pradžia](./00-course-setup/README.md).
