# നോട്ട്‌ബുക്ക് ഘടന

Jupyter നോട്ട്ബുക്കുകൾ ഈ ഉയർന്ന തലത്തിലുള്ള ഘടനയുള്ള JSON രേഖകളാണ്:

- `nbformat` and `nbformat_minor`
- `metadata`
- `cells` (markdown-ഉം കോഡ് സെല്ലുകളും ഉള്ള ഒരു പട്ടിക)

When editing `.ipynb` files programmatically:

- ടെംപ്ലേറ്റിൽ നിന്നുള്ള `nbformat` மற்றும் `nbformat_minor` സംരക്ഷിക്കുക.
- `cells`-നെ ക്രമത്തിൽ ഉള്ള പട്ടികയായിരിക്കണം; ഉദ്ദേശപൂര്‍വമല്ലെങ്കിൽ ക്രമം മാറ്റരുത്.
- കോഡ് സെല്ലുകൾക്കായി, അറിയാനാകാത്തപ്പോള്‍ `execution_count`-നെ `null` ആയി സജ്ജമാക്കുക.
- കോഡ് സെല്ലുകൾക്കായി, തയാറാക്കുമ്പോൾ `outputs`-നെ ശൂന്യ പട്ടികയായി സജ്ജമാക്കുക.
- മാർക്ക്ഡൗൺ സെല്ലുകൾക്കായി, `cell_type="markdown"`യും `metadata={}`യും നിലനിർത്തുക.

Raw notebook JSON കൈത്തയ്യാറാക്കി നേരിട്ട് എഴുതുന്നതിനുള്ള പകരമായി, ബണ്ടിൽ ചെയ്ത ടെംപ്ലേറ്റുകൾ അല്ലെങ്കിൽ `new_notebook.py` (ഉദാഹരണത്തിന്, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) എന്നിവയിൽ നിന്നുള്ള scaffolding ഉപയോഗിക്കാൻ മുൻഗണന നൽകുക.
