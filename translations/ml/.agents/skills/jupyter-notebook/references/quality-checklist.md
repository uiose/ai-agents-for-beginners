# ഗുണനിലവാര പരിശോധനാ പട്ടിക

ഒരു നോട്ട്‌ബുക്ക് നൽകുന്നതിന് മുമ്പ്:

- അതിനെ മുകളിലുനിന്ന് താഴേക്ക് കുറഞ്ഞത് ഒരു തവണ പ്രവർത്തിപ്പിക്കുക (അല്ലെങ്കിൽ പരിസ്ഥിതി അനുവദിക്കുന്നതുവരെ).
- ആദ്യ സെല്ലുകൾ ആവശ്യമായ എല്ലാ അവസ്ഥകളും സജ്ജമാക്കുന്നുണ്ടെന്ന് ഉറപ്പാക്കുക; മുൻ റൺകളിൽ നിന്നുള്ള മറഞ്ഞ അവസ്ഥ ഒഴിവാക്കുക.
- ഔട്ട്പുട്ടുകൾ ക്രമത്തിൽ സൂക്ഷിക്കുക. ചെറിയൊരു സംഗ്രഹം വേണ്ടപ്പോൾ വലിയ ഔട്ട്പുട്ടുകൾ ഒഴിവാക്കുക.
- ചെറിയ പട്ടികകൾ, പ്രധാന മാപനങ്ങൾ, അല്ലെങ്കിൽ ചുരുങ്ങിയ പ്രിന്റ് ഔട്ടുകൾ പ്രധാനം ചെയ്യും.
- വിവരണം എളുപ്പത്തിൽ സ്കാൻ ചെയ്യാവുന്ന രീതിയിൽ സജ്ജമാക്കുക. തലക്കെട്ടുകളും ചെറിയ ബുള്ളറ്റുകളും ഉപയോഗിക്കുക, ദീർഘമായ അനുഛേദങ്ങൾ ഒഴിവാക്കുക.
- ആവശ്യമായപ്പോൾ മാത്രമേ സഹായകരമായ TODOകൾ വിടാവാൻ അനുവദിക്കുക, അവ വ്യക്തമായി ലേബൽ ചെയ്യുക.
- പ്രവർത്തനം സാധ്യമല്ലെങ്കിൽ, അപകട സാധ്യതയും പ്രാദേശികമായി എങ്ങനെ സാധൂകരിക്കാമെന്നുമെന്തെന്നു വ്യക്തമാക്കുക.
