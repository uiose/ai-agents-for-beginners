# AGENTS.md

## പ്രോജക്ട് അവലോകനം

ഈ റിപോസിറ്ററി "AI Agents for Beginners" എന്ന കോഴ്സ് ഉൾക്കൊണ്ടിരിക്കുന്നു - AI ഏജന്റുകൾ നിർമ്മിക്കുന്നതിനുള്ള അടിസ്ഥാനങ്ങൾ മുതൽ ഡിസൈൻ പാറ്റേണുകൾ, ഫ്രെയിംവർക്കുകൾ, പ്രൊഡക്ഷൻ ഡിപ്ലോയ്മെന്റ് വരെ എല്ലാം പഠിപ്പിക്കുന്ന സമഗ്ര വിദ്യാഭ്യാസ കോഴ്സ്. കോഴ്സ് 15+ പാഠങ്ങളടങ്ങിയതായി ഫണ്ടമെന്റലുകൾ, ഡിസൈൻ പാറ്റേണുകൾ, ഫ്രെയിംവർക്കുകൾ, மற்றும் AI ഏജന്റുകളുടെ പ്രൊഡക്ഷൻ ഡിപ്ലോയ്മെന്റ് ഉൾക്കൊള്ളുന്നവയാണ്.

**പ്രധാന സാങ്കേതിക വിദ്യകൾ:**
- Python 3.12+
- ഇന്ററാക്ടിവ് പഠനത്തിന് Jupyter നോട്ട്‌ബുക്കുകൾ
- എഐ ഫ്രെയിംവർക്കുകൾ: Microsoft Agent Framework (MAF)
- Azure AI സേവനങ്ങൾ: Microsoft Foundry, Azure AI Foundry Agent Service V2

**ആർക്കിടെക്ചർ:**
- പാഠഭാഗം അടിസ്ഥാനമായ ഘടന (00-15+ ഡയറക്ടറികൾ)
- ഓരോ പാഠത്തിലും ഉണ്ട്: README ഡോക്യുമെന്റേഷൻ, കോഡ് സാമ്പിൾസ് (Jupyter നോട്ട്‌ബുക്കുകൾ), ছবികൾ
- ഓട്ടോമേറ്റഡ് വിവർത്തന സംവിധാനത്തിന്റെ മുഖേന ബഹുഭാഷാ പിന്തുണ
- Microsoft Agent Framework ഉപയോഗിച്ച് ഓരോ പാഠത്തിലും ഒന്ന് Python നോട്ട്‌ബുക്കുകൾ

## സെറ്റ്‌അപ്പ് കമാൻഡുകൾ

### ആവശ്യമായതുകൾ
- Python 3.12 അല്ലെങ്കിൽ അതിനുതന്നെമുകളിലുള്ളത്
- Azure subscription (Azure AI Foundry-ന്)
- ഇൻസ്റ്റാൾ ചെയ്തു തുറന്നിരിക്കുന്ന Azure CLI (`az login`)

### പ്രാഥമിക സെറ്റ്‌അപ്പ്

1. **റിപോസിറ്ററി ക്ലോൺ ചെയ്യുക അല്ലെങ്കിൽ ഫോർക്ക് ചെയ്യുക:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # അതവാ
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python വെർച്ച്വൽ എൻവയോൺമെന്റ് സൃഷ്ടിച്ച് ആക്ടിവേറ്റ് ചെയ്യുക:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Windows-ൽ: venv\Scripts\activate
   ```

3. **ഡിപ്പൻഡൻസികൾ ഇൻസ്റ്റാൾ ചെയ്യുക:**
   ```bash
   pip install -r requirements.txt
   ```

4. **പരിസ്ഥിതി വേരിയബിളുകൾ സജ്ജീകരിക്കുക:**
   ```bash
   cp .env.example .env
   # നിങ്ങളുടെ API കീകളും എൻഡ്‌പോയിന്റുകളും ഉൾപ്പെടുത്തി .env ഫയൽ തിരുത്തുക
   ```

### ആവശ്യമായ പരിസ്ഥിതി വേരിയബിളുകൾ

**Azure AI Foundry**-ക്കായി (ആവശ്യമാണ്):
- `AZURE_AI_PROJECT_ENDPOINT` - Azure AI Foundry പ്രോജക്ട് എEndpoint
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - മോഡൽ ഡിപ്പ്ലോയ്‌മെന്റ് നാമം (ഉദാഹരണം: gpt-4o)

**Azure AI Search**-ക്കായി (പാഠം 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search എEndpoint
- `AZURE_SEARCH_API_KEY` - Azure AI Search API കീ

ഓथന്റിക്കേഷൻ: നോട്ട്‌ബുക്കുകൾ ഓടിക്കുന്നതിന് മുമ്പ് `az login` ഓടിക്കുക (ഉപയോജിക്കുന്നു `AzureCliCredential`).

## ഡെവലപ്മെന്റ് വർക്ക്‌ഫ്ലോ

### Jupyter നോട്ട്‌ബുക്കുകൾ ഓടിക്കൽ

ഓരോ പാഠത്തിലും വിവിധ ഫ്രെയിംവർകുകൾക്കുള്ള നിരവധി Jupyter നോട്ട്‌ബുക്കുകൾ ഉണ്ട്:

1. **Jupyter ആരംഭിക്കുക:**
   ```bash
   jupyter notebook
   ```

2. **ഒരു പാഠം ഡയറക്ടറിയിലേക്ക് നാവിഗേറ്റ് ചെയ്യുക** (ഉദാ., `01-intro-to-ai-agents/code_samples/`)

3. **നോട്ട്‌ബുക്കുകൾ തുറന്ന് ഓടിക്കുക:**
   - `*-python-agent-framework.ipynb` - Microsoft Agent Framework (Python) ഉപയോഗിച്ച്
   - `*-dotnet-agent-framework.ipynb` - Microsoft Agent Framework (.NET) ഉപയോഗിച്ച്

### Microsoft Agent Framework ഉപയോഗിച്ച് ജോലി ചെയ്യുന്നത്

**Microsoft Agent Framework + Azure AI Foundry:**
- Azure subscription ആവശ്യമാണ്
- Agent Service V2-ക്കായി `AzureAIProjectAgentProvider` ഉപയോഗിക്കുന്നു (Foundry പോർട്ടലിൽ ഏജന്റുകൾ ദൃശ്യമാകുന്നു)
- നിര്‍മ്മാണത്തിന് പ്രിയപ്പെട്ടതും ഇൻബിൽട്ട് ഓബ്സർവബിലിറ്റി ഉള്ളതും
- ഫയൽ പാറ്റേൺ: `*-python-agent-framework.ipynb`

   - വ്യക്തിഗത പാക്കേജുകൾ ഇൻസ്റ്റാൾ ചെയ്യുന്നത് ഒഴിവാക്കി `requirements.txt`-ൽ നിന്നാണ് ഇൻസ്റ്റാൾ ചെയ്യുക
