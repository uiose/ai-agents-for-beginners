# Github MCP സർവർ ഉദാഹരണം

## വിവരണം

ഈ ഡെമോ മൈക്രോസോഫ്റ്റ് റിയാക്ടർ വഴി നടത്തിയ AI ഏജൻറ്സ് ഹാക്കാഥോണിനായി സൃഷ്‌ടിച്ച ഒരു ഡെമോയാണ്.

ഉപകരണങ്ങൾ ഉപയോഗിച്ച് ഒരു ഉപയോക്താവിന്റെ Github റിപൊസികളെ അടിസ്ഥാനമാക്കി ഹാക്കാഥോൺ പ്രൊജക്ടുകൾ ശുപാർശ ചെയ്യുന്നത് ആണ് ഈ ഡെമോയുടെ പ്രയോജനം.  
ഇത് നടപ്പാക്കുന്നത്:

1. **Github ഏജന്റ്** - Github MCP സർവർ ഉപയോഗിച്ച് റിപൊസുകളും ആ റിപൊസികളുമായി ബന്ധപ്പെട്ട വിവരങ്ങളും പിടിച്ചെടുക്കാൻ.
2. **ഹാക്കാഥോൺ ഏജന്റ്** - Github ഏജന്റിൽ നിന്നുള്ള ഡാറ്റ վերցിച്ച്, ഉപയോക്താവിന്റെ പ്രോജക്ടുകൾ, ഉപയോഗിച്ച ഭാഷകൾ, AI ഏജൻറ്സ് ഹാക്കാഥോണിലെ പ്രോജക്ട് ട്രാക്കുകൾ അടിസ്ഥാനമാക്കി സൃഷ്‌ടിപരമായ ഹാക്കാഥോൺ പ്രോജക്ട് ആശയങ്ങൾ ഉണ്ടാക്കുന്നു.
3. **ഇവന്റ്‌സ് ഏജന്റ്** - ഹാക്കാഥോൺ ഏജന്റിന്റെ നിർദ്ദേശം അടിസ്ഥാനമാക്കി, AI ഏജന്റ് ഹാക്കാഥോൺ സീരിസിൽ നിന്നുള്ള ബന്ധപ്പെട്ട ഇവന്റുകൾ ശുപാർശ ചെയ്യും.  

## കോഡ് പ്രവർത്തിപ്പിക്കൽ

### പരിസ്ഥിതി വ്യത്യസ്ഥങ്ങൾ

ഈ ഡെമോ Microsoft Agent Framework, Azure OpenAI Service, Github MCP Server, Azure AI Search എന്നിവ ഉപയോഗിക്കുന്നു.

ഈ ഉപകരണങ്ങൾ ഉപയോഗിക്കാൻ നിങ്ങൾക്കു വേണ്ടുന്ന ശരിയായ പരിസ്ഥിതി വ്യത്യസ്ഥങ്ങൾ സജ്ജമാക്കിയിരിക്കണം:

```python
AZURE_AI_PROJECT_ENDPOINT=""
AZURE_AI_MODEL_DEPLOYMENT_NAME=""
AZURE_SEARCH_SERVICE_ENDPOINT=""
AZURE_SEARCH_API_KEY=""
``` 


## Chainlit സർവർ പ്രവർത്തിപ്പിക്കൽ

MCP സർവറുമായി ബന്ധിപ്പിക്കാൻ, ഈ ഡെമോ Chainlit നെ ചാറ്റ് ഇൻറർഫെയ്സ് ആയി ഉപയോഗിക്കുന്നു. 

സർവർ പ്രവർത്തിപ്പിക്കാൻ നിങ്ങളുടെ ടെർമിനലിൽ താഴെയുള്ള കമാൻഡ് ഉപയോഗിക്കുക:

```bash
chainlit run app.py -w
```


ഇത് നിങ്ങളുടെ Chainlit സർവർ `localhost:8000` ൽ ആരംഭിക്കും കൂടാതെ `event-descriptions.md` ഉള്ളടക്കം ഉപയോഗിച്ച് Azure AI Search ഇൻഡക്സ് സമൃദ്ധിപ്പെടുത്തും.  

## MCP സർവറുമായി ബന്ധിപ്പിക്കൽ

Github MCP സർവറുമായി ബന്ധിപ്പിക്കാൻ, "Type your message here.." ചാറ്റ് ബോക്സിന്റെ താഴെ ഉള്ള "plug" ഐകൺ തിരഞ്ഞെടുക്കുക:

![MCP Connect](../../../../../translated_images/ml/mcp-chainlit-1.7ed66d648e3cfb28.webp)

അവിടെ നിന്നു "Connect an MCP" എന്നതിൽ ക്ലിക്കുചെയ്ത് Github MCP Server ബന്ധിപ്പിക്കാൻ കമാൻഡ് ചേർക്കാം:

```bash
npx -y @modelcontextprotocol/server-github --env GITHUB_PERSONAL_ACCESS_TOKEN=[YOUR PERSONAL ACCESS TOKEN]
```
  
"[YOUR PERSONAL ACCESS TOKEN]" എന്ന് ഉള്ളത് നിങ്ങളുടെ യഥാർത്ഥ Personal Access Token ഉപയോഗിച്ച് പകരുക.

ബന്ധിപ്പിച്ചതിനു ശേഷം, കണക്റ്റഡ് ആണെന്ന് സ്ഥിരീകരിക്കാൻ പ്ലഗ് ഐകണിന്റെ അടുത്ത് (1) കാണണം. കാണാതിരുന്നാൽ `chainlit run app.py -w` കമാൻഡ് ഉപയോഗിച്ച് chainlit സർവർ റീസ്റ്റാർട്ട് ചെയ്യുക.

## ഡെമോ ഉപയോഗിക്കൽ

ഹാക്കാഥോൺ പ്രോജക്ടുകൾ ശുപാർശ ചെയ്യുന്നതിനുള്ള ഏജന്റ് പ്രവർത്തനക്രമം ആരംഭിക്കാൻ നിങ്ങൾക്ക് ഇങ്ങനെ ഒരു സന്ദേശം ടൈപ്പ് ചെയ്യാം:

"Recommend hackathon projects for the Github user koreyspace"

റൂട്ടർ ഏജന്റ് നിങ്ങളുടെ അഭ്യർത്ഥന വിശകലനം ചെയ്ത് ഏത് ഏജന്റുമാരുടെ (GitHub, Hackathon, Events) സംയോജനം നിങ്ങളുടെ ചോദ്യം കൈകാര്യം ചെയ്യുന്നതിന് ഏറ്റവും അനുയോജ്യമാണ് എന്ന് തീരുമാനിക്കും. ഈ ഏജന്റുമാർGitHub റിപോസിറ്റി വിശകലനം, പ്രോജക്ട് ആശയവൽക്കരണം, ബന്ധപ്പെട്ട ടെക് ഇവന്റുകൾ എന്നിവ അടിസ്ഥാനമാക്കി സമഗ്രമായ ശുപാർശകൾ നൽകുന്നതിന് ഒരുമിച്ച് പ്രവർത്തിക്കും.
