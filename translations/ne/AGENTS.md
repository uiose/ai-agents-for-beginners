# AGENTS.md

## परियोजना अवलोकन

यस रिपोजिटरीमा "शुरुआतीहरूका लागि AI एजेन्टहरू" समावेश छन् - एक समग्र शैक्षिक पाठ्यक्रम जसले AI एजेन्टहरू बनाउन आवश्यक सबै कुरा सिकाउँछ। यो पाठ्यक्रममा 15+ पाठहरू छन् जसमा आधारभूत कुरा, डिजाइन ढाँचा, फ्रेमवर्कहरू, र AI एजेन्टहरूको उत्पादन तैनाती समावेश छन्।

**मुख्य प्रविधिहरू:**
- Python 3.12+
- अन्तरक्रियात्मक सिकाइका लागि Jupyter नोटबुकहरू
- AI फ्रेमवर्कहरू: Microsoft Agent Framework (MAF)
- Azure AI सेवाहरू: Microsoft Foundry, Azure AI Foundry Agent Service V2

**वास्तुकला:**
- पाठ-आधारित संरचना (00-15+ निर्देशिकाहरू)
- प्रत्येक पाठमा हुन्छ: README कागजात, कोड नमूना (Jupyter नोटबुकहरू), र चित्रहरू
- स्वचालित अनुवाद प्रणालीमार्फत बहुभाषी समर्थन
- प्रत्येक पाठमा Microsoft Agent Framework प्रयोग गरेर एउटा Python नोटबुक

## सेटअप आदेशहरू

### आवश्यकताहरू
- Python 3.12 वा माथि
- Azure सदस्यता (Azure AI Foundry का लागि)
- Azure CLI स्थापना र प्रमाणित (`az login`)

### प्रारम्भिक सेटअप

1. **रिपोजिटरी क्लोन वा फोर्क गर्नुहोस्:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # वा
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python भर्चुअल वातावरण सिर्जना र सक्रिय गर्नुहोस्:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # विन्डोजमा: venv\Scripts\activate
   ```

3. **निर्भरता स्थापना गर्नुहोस्:**
   ```bash
   pip install -r requirements.txt
   ```

4. **पर्यावरण चरहरू सेटअप गर्नुहोस्:**
   ```bash
   cp .env.example .env
   # आफ्नो API कुञ्जीहरू र अन्तबिन्दुहरू सहित .env सम्पादन गर्नुहोस्
   ```

### आवश्यक वातावरण चरहरू

**Azure AI Foundry** का लागि (आवश्यक):
- `AZURE_AI_PROJECT_ENDPOINT` - Azure AI Foundry प्रोजेक्ट अन्तबिन्दु
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - मोडेल तैनाती नाम (जस्तै gpt-4o)

**Azure AI Search** का लागि (पाठ 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search अन्तबिन्दु
- `AZURE_SEARCH_API_KEY` - Azure AI Search API कुञ्जी

प्रमाणिकरण: नोटबुकहरू चलाउनु अघि `az login` चलाउनुहोस् (यसले `AzureCliCredential` प्रयोग गर्दछ)।
