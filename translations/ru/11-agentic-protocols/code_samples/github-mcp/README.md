# Github MCP Server Example

## Описание

Это демонстрация, созданная для AI Agents Hackathon, организованного через Microsoft Reactor.

Инструмент используется для рекомендации проектов хакатона на основе репозиториев пользователя в Github.  
Это достигается с помощью:

1. **Github Agent** - Использование Github MCP Server для получения репозиториев и информации о них.
2. **Hackathon Agent** - Принимает данные от Github Agent и генерирует креативные идеи проектов для хакатона на основе проектов, языков, используемых пользователем, и треков проектов хакатона AI Agents.
3. **Events Agent** - Исходя из предложений hackathon agent, events agent рекомендует релевантные события из серии AI Agent Hackathon.

## Запуск кода

### Переменные окружения

В этой демонстрации используется Microsoft Agent Framework, Azure OpenAI Service, Github MCP Server и Azure AI Search.

Убедитесь, что у вас установлены правильные переменные окружения для работы с этими инструментами:

```python
AZURE_AI_PROJECT_ENDPOINT=""
AZURE_AI_MODEL_DEPLOYMENT_NAME=""
AZURE_SEARCH_SERVICE_ENDPOINT=""
AZURE_SEARCH_API_KEY=""
``` 


## Запуск Chainlit сервера

Для подключения к MCP серверу в этой демонстрации используется Chainlit в качестве интерфейса чата.

Для запуска сервера выполните следующую команду в терминале:

```bash
chainlit run app.py -w
```


Это запустит ваш Chainlit сервер на `localhost:8000`, а также заполнит ваш индекс Azure AI Search содержимым файла `event-descriptions.md`.

## Подключение к MCP серверу

Чтобы подключиться к Github MCP Server, нажмите на иконку "штекера" под полем ввода текста "Type your message here..":

![MCP Connect](../../../../../translated_images/ru/mcp-chainlit-1.7ed66d648e3cfb28.webp)

Далее нажмите на "Connect an MCP", чтобы добавить команду подключения к Github MCP Server:

```bash
npx -y @modelcontextprotocol/server-github --env GITHUB_PERSONAL_ACCESS_TOKEN=[YOUR PERSONAL ACCESS TOKEN]
```
  
Замените "[YOUR PERSONAL ACCESS TOKEN]" на ваш реальный персональный токен доступа.

После подключения вы должны увидеть цифру (1) рядом с иконкой штекера, что подтверждает соединение. Если нет, попробуйте перезапустить chainlit сервер командой `chainlit run app.py -w`.

## Использование демо

Чтобы запустить рабочий процесс агента по рекомендации проектов для хакатона, вы можете написать сообщение, например:

"Recommend hackathon projects for the Github user koreyspace"

Router Agent проанализирует ваш запрос и определит, какая комбинация агентов (GitHub, Hackathon и Events) лучше всего подходит для обработки вашего запроса. Агенты работают вместе, чтобы предоставить комплексные рекомендации на основе анализа репозиториев Github, идей проектов и соответствующих технических событий.
