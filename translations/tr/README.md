>
> **CMD (Windows):**

> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git

## 🌱 Başlarken

Bu kurs, Yapay Zeka Ajanları oluşturmanın temel prensiplerini içeren dersler sunar. Her ders kendi konusunu kapsar, bu yüzden istediğiniz yerden başlayabilirsiniz!

Bu kurs için çoklu dil desteği mevcuttur. Mevcut diller için [buraya bakabilirsiniz](../..).

Eğer Üretken Yapay Zeka modelleri ile ilk kez çalışıyorsanız, 21 dersi kapsayan [Yeni Başlayanlar için Üretken Yapay Zeka](https://aka.ms/genai-beginners) kursumuza göz atın.

Bu depoyu [yıldızlamayı (🌟) ve](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ve [çatallamayı (fork)](https://github.com/microsoft/ai-agents-for-beginners/fork) unutmayın, böylece kodu çalıştırabilirsiniz.

### Diğer Öğrencilerle Tanışın, Sorularınızı Cevaplayın

Yapay Zeka Ajanları geliştirme konusunda takılırsanız veya sorularınız varsa, [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) içinde özel Discord Kanalımıza katılabilirsiniz.

### Gerekenler

Bu kurstaki her dersin kod örnekleri vardır ve bunlar code_samples klasöründe bulunabilir. Kendi kopyanızı oluşturmak için [bu depoyu çatallayabilirsiniz (fork)](https://github.com/microsoft/ai-agents-for-beginners/fork).

Bu alıştırmalardaki kod örnekleri Microsoft Agent Framework ile Azure AI Foundry Agent Service V2 kullanılmaktadır:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure Hesabı Gereklidir

Bu kurs, Microsoft'un aşağıdaki Yapay Zeka Ajanı çerçevelerini ve servislerini kullanır:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Kurs için kodu çalıştırma hakkında daha fazla bilgiye [Kurs Kurulumu](./00-course-setup/README.md) sayfasından ulaşabilirsiniz.
