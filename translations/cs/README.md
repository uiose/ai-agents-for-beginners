## 🌱 Začínáme

Tento kurz obsahuje lekce pokrývající základy tvorby AI Agentů. Každá lekce se věnuje vlastnímu tématu, takže začněte kdekoliv chcete!

Kurz má vícejazyčnou podporu. Podívejte se na naše [dostupné jazyky zde](../..).

Pokud je to váš první pokus o tvorbu s generativními AI modely, podívejte se na náš kurz [Generative AI For Beginners](https://aka.ms/genai-beginners), který obsahuje 21 lekcí o tvorbě s GenAI.

Nezapomeňte [označit (🌟) tento repozitář hvězdou](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) a [vytvořit fork tohoto repozitáře](https://github.com/microsoft/ai-agents-for-beginners/fork) pro spuštění kódu.

### Poznejte další studenty, nechte si zodpovědět své otázky

Pokud se zaseknete nebo máte jakékoli otázky ohledně tvorby AI Agentů, připojte se do našeho speciálního kanálu na Discordu v [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Co potřebujete

Každá lekce v tomto kurzu obsahuje ukázky kódu, které najdete ve složce code_samples. Můžete si [vytvořit fork tohoto repozitáře](https://github.com/microsoft/ai-agents-for-beginners/fork), abyste měli vlastní kopii.

Ukázky kódu v těchto cvičeních využívají Microsoft Agent Framework s Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Požadován účet Azure

Tento kurz používá následující AI Agent frameworky a služby od Microsoftu:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Pro více informací o spuštění kódu tohoto kurzu přejděte na [Course Setup](./00-course-setup/README.md).
