## 🌱 شروع به کار

این دوره درس‌هایی را پوشش می‌دهد که اصول ساخت عامل‌های هوش مصنوعی را آموزش می‌دهد. هر درس موضوع خود را دارد پس از هر جایی که دوست دارید شروع کنید!

پشتیبانی چند زبان برای این دوره وجود دارد. به [زبان‌های موجود در اینجا](../..) مراجعه کنید.

اگر این اولین بار است که با مدل‌های هوش مصنوعی مولد کار می‌کنید، دوره [هوش مصنوعی مولد برای مبتدیان](https://aka.ms/genai-beginners) ما را بررسی کنید که شامل ۲۱ درس در مورد ساخت با GenAI است.

فراموش نکنید که به این مخزن [ستاره بدهید (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) و [این مخزن را فورک کنید](https://github.com/microsoft/ai-agents-for-beginners/fork) تا کد را اجرا کنید.

### ملاقات با دیگر زبان‌آموزان، پاسخ به سوالات خود را دریافت کنید

اگر گیر کردید یا سوالی درباره ساخت عامل‌های هوش مصنوعی داشتید، به کانال اختصاصی ما در دیسکورد [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) بپیوندید.

### آنچه نیاز دارید

هر درس در این دوره نمونه‌های کد دارد که می‌توانید در پوشه code_samples بیابید. می‌توانید [این مخزن را فورک کنید](https://github.com/microsoft/ai-agents-for-beginners/fork) تا کپی خود را بسازید.

نمونه‌های کد در این تمرین‌ها از Microsoft Agent Framework با سرویس Azure AI Foundry Agent V2 استفاده می‌کنند:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - نیاز به حساب Azure

این دوره از چارچوب‌ها و سرویس‌های عامل هوش مصنوعی زیر از مایکروسافت استفاده می‌کند:

- [چارچوب عامل مایکروسافت (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [سرویس Azure AI Foundry Agent V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

برای اطلاعات بیشتر درباره اجرای کد این دوره به [تنظیمات دوره](./00-course-setup/README.md) مراجعه کنید.
