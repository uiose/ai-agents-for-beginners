## 🌱 ਸ਼ੁਰੂਆਤ

ਇਸ ਕੋਰਸ ਵਿੱਚ AI ਏਜੰਟ ਬਣਾਉਣ ਦੇ ਮੁਢਲੇ ਸਿਧਾਂਤਾਂ ਤੇ ਪਾਠ ਹਨ। ਹਰ ਪਾਠ ਵਿੱਚ ਆਪਣਾ ਵਿਸ਼ਾ ਹੈ, ਤਾਂ ਜੋ ਤੁਸੀਂ ਇੱਥੇ ਕਿਤੇ ਤੋਂ ਵੀ ਸ਼ੁਰੂ ਕਰ ਸਕੋ!

ਇਸ ਕੋਰਸ ਲਈ ਬਹੁ-ਭਾਸ਼ਾ ਸਹਾਇਤਾ ਉਪਲਬਧ ਹੈ। ਸਾਡੀ [ਉਪਲਬਧ ਭਾਸ਼ਾਵਾਂ](../..) 'ਤੇ ਜਾਓ।

ਜੇ ਤੁਸੀਂ ਪਹਿਲੀ ਵਾਰ ਜਨੇਰੇਟਿਵ AI ਮਾਡਲ ਨਾਲ ਕੰਮ ਕਰ ਰਹੇ ਹੋ, ਤਾਂ ਸਾਡਾ [ਜਨੇਰੇਟਿਵ AI ਸ਼ੁਰੂਆਤੀ ਲਈ](https://aka.ms/genai-beginners) ਕੋਰਸ ਵੇਖੋ, ਜਿਸ ਵਿੱਚ GenAI ਨਾਲ ਬਣਾਉਣ ਦੇ 21 ਪਾਠ ਹਨ।

ਭੁੱਲੋ ਨਾ ਕਿ [ਇਸ ਰਿਪੋ ਨੂੰ ਸਿਤਾਰਾ (🌟) ਦਿਓ](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ਅਤੇ [ਇਸ ਰਿਪੋ ਨੂੰ ਫੋਰਕ ਕਰੋ](https://github.com/microsoft/ai-agents-for-beginners/fork) ਤਾਂ ਜੋ ਕੋਡ ਚਲਾਇਆ ਜਾ ਸਕੇ।

### ਹੋਰ ਸਿਖਿਆਰਥੀਆਂ ਨਾਲ ਮਿਲੋ, ਆਪਣੇ ਸਵਾਲ ਪੁੱਛੋ

ਜੇ ਤੁਸੀਂ ਫਸ ਜਾਂਦੇ ਹੋ ਜਾਂ AI ਏਜੰਟ ਬਣਾਉਣ ਬਾਰੇ ਕੋਈ ਵੀ ਸਵਾਲ ਹੈ, ਤਾਂ ਸਾਡੀ ਫੌਕਸਡ Discord ਚੈਨਲ ਵਿੱਚ ਸ਼ਾਮਲ ਹੋਵੋ [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) ਵਿੱਚ।

### ਤੁਹਾਨੂੰ ਕੀ ਚਾਹੀਦਾ ਹੈ

ਇਸ ਕੋਰਸ ਦੇ ਹਰ ਪਾਠ ਵਿੱਚ ਕੋਡ ਉਦਾਹਰਣਾਂ ਹੁੰਦੀਆਂ ਹਨ, ਜੋ code_samples ਫੋਲਡਰ ਵਿੱਚ ਮਿਲਦੀਆਂ ਹਨ। ਤੁਸੀਂ ਆਪਣੀ ਕਾਪੀ ਬਣਾਉਣ ਲਈ [ਇਸ ਰਿਪੋ ਨੂੰ ਫੋਰਕ ਕਰ ਸਕਦੇ ਹੋ](https://github.com/microsoft/ai-agents-for-beginners/fork)।

ਇਹ ਕੋਡ ਉਦਾਹਰਣਾਂ Microsoft Agent Framework ਨਾਲ Azure AI Foundry Agent Service V2 ਦੀ ਵਰਤੋਂ ਕਰਦੀਆਂ ਹਨ:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure ਖਾਤਾ ਲਾਜ਼ਮੀ

ਇਸ ਕੋਰਸ ਵਿੱਚ Microsoft ਵੱਲੋਂ ਹੇਠਾਂ ਦਿੱਤੇ AI ਏਜੰਟ ਫਰੇਮਵਰਕ ਅਤੇ ਸੇਵਾਵਾਂ ਦੀ ਵਰਤੋਂ ਕੀਤੀ ਗਈ ਹੈ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

ਕੋਰਸ ਦੇ ਲਈ ਕੋਡ ਚਲਾਉਣ ਬਾਰੇ ਹੋਰ ਜਾਣਕਾਰੀ ਲਈ, [Course Setup](./00-course-setup/README.md) 'ਤੇ ਜਾਓ।
