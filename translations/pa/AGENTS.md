## ਸੈੱਟਅੱਪ ਕਮਾਂਡਜ਼

### ਪਹਿਲਾਂ ਦੀਆਂ ਲੋੜਾਂ
- ਪਾਇਥਨ 3.12 ਜਾਂ ਉਸ ਤੋਂ ਉੱਚਾ
- ਅਜ਼ੂਰ سبਸਕ੍ਰਿਪਸ਼ਨ (ਅਜ਼ੂਰ ਏਆਈ ਫਾਊਂਡਰੀ ਲਈ)
- ਅਜ਼ੂਰ CLI ਇੰਸਟਾਲ ਅਤੇ ਪ੍ਰਮਾਣਿਤ (`az login`)

### ਸ਼ੁਰੂਆਤੀ ਸੈੱਟਅੱਪ

1. **ਰੇਪੋਜਿਟਰੀ ਨੂੰ ਕਲੋਨ ਜਾਂ ਫੋਰਕ ਕਰੋ:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # ਜਾਂ
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **ਪਾਇਥਨ ਵਰਚੁਅਲ ਐਨਵਾਇਰਨਮੈਂਟ ਬਣाओ ਅਤੇ ਸક્રਿਯ ਕਰੋ:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # ਵਿੰਡੋਜ਼ 'ਤੇ: venv\Scripts\activate
   ```

3. **ਡਿਪੈਂਡਿਸੀਆਂ ਇੰਸਟਾਲ ਕਰੋ:**
   ```bash
   pip install -r requirements.txt
   ```

4. **ਐਨਵਾਇਰਨਮੈਂਟ ਵਰਿਏਬਲ ਸੈੱਟ ਕਰੋ:**
   ```bash
   cp .env.example .env
   # ਆਪਣੇ API ਕੁੰਜੀਆਂ ਅਤੇ ਐਂਡਪੌਇੰਟਸ ਨਾਲ .env ਸੋਧੋ
   ```

### ਲੋੜੀਂਦੇ ਐਨਵਾਇਰਨਮੈਂਟ ਵਰਿਏਬਲ

**ਅਜ਼ੂਰ ਏਆਈ ਫਾਊਂਡਰੀ** ਲਈ (ਲੋੜੀਂਦਾ):
- `AZURE_AI_PROJECT_ENDPOINT` - ਅਜ਼ੂਰ ਏਆਈ ਫਾਊਂਡਰੀ ਪ੍ਰੋਜੈਕਟ ਐਂਡਪੋਇੰਟ
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - ਮਾਡਲ ਡਿਪਲੋਇਮੈਂਟ ਦਾ ਨਾਮ (ਜਿਵੇਂ gpt-4o)

**ਅਜ਼ੂਰ ਏਆਈ ਖੋਜ (ਲੈਸਨ 05 - RAG):**
- `AZURE_SEARCH_SERVICE_ENDPOINT` - ਅਜ਼ੂਰ ਏਆਈ ਖੋਜ ਐਂਡਪੋਇੰਟ
- `AZURE_SEARCH_API_KEY` - ਅਜ਼ੂਰ ਏਆਈ ਖੋਜ API ਕੁੰਜੀ

ਪ੍ਰਮਾਣਿਕਤਾ: ਨੋਟਬੁਕ ਚਲਾਉਣ ਤੋਂ ਪਹਿਲਾਂ `az login` ਚਲਾਓ (`AzureCliCredential` ਵਰਤਦਾ ਹੈ)।
