## 🌱 Alustamine

Selles kursuses on õppetunnid, mis käsitlevad AI-agentide loomise põhialuseid. Iga õppetund keskendub oma teemale — alusta sealt, kus soovid!

Selle kursuse jaoks on mitmekeelne tugi. Vaata meie [saadaval olevaid keeli siit](../..). 

Kui see on sinu esimene kord genereerivate tehisintellektimudelitega ehitamisel, vaata meie [Generative AI For Beginners](https://aka.ms/genai-beginners) kursust, mis sisaldab 21 õppetundi GenAI-ga ehitamise kohta.

Ära unusta [märgistada seda repositooriumi tärniga (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ning [forkida see repo](https://github.com/microsoft/ai-agents-for-beginners/fork), et koodi käivitada.

### Kohtu teiste õppijatega, saa oma küsimustele vastused

Kui jääd hätta või sul on küsimusi AI-agentide ehitamise kohta, liitu meie pühendatud Discord-kanaliga [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Mida sul vaja on

Iga kursuse õppetund sisaldab koodinäiteid, mis asuvad code_samples kaustas. Sa võid [forkida selle repo](https://github.com/microsoft/ai-agents-for-beginners/fork), et luua oma koopia.  

Neis harjutustes kasutatavad koodinäited kasutavad Microsoft Agent Frameworki koos Azure AI Foundry Agent Service V2-teenusega:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - nõuab Azure'i kontot

See kursus kasutab Microsofti järgmisi AI-agentide raamistikke ja teenuseid:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Lisainfo saamiseks kursuse koodi käivitamise kohta vaata [Course Setup](./00-course-setup/README.md).
