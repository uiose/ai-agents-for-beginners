## 🌱 ప్రారంభించడం

ఈ కోర్సులో AI ఏజెంట్లు నిర్మాణం మూలాలు బోధించటానికి పాఠాలు ఉన్నాయి. ప్రతి పాఠం తాను చర్చించే విషయం మీదే ఉంటుంది కాబట్టి మీరు ఇష్టమైన దిక్కులోనైనా మొదలుపెట్టవచ్చు!

ఈ కోర్సుకు బహుభాషా మద్దతు ఉంది. మా [లభ్యమయ్యే భాషలు ఇక్కడ](../..) చూడండి.

మీరు మొదటిసారి జనరేటివ్ AI నమూనాలతో పని చేస్తున్నట్లయితే, మా [ప్రారంభికుల కోసం జనరేటివ్ AI](https://aka.ms/genai-beginners) కోర్సును చూడండి, ఇది 21 పాఠాలతో GenAI తో ఎలా నిర్మించాలో చెప్పబడింది.

ఈ రిపోజిటరీకు [స్టార్ (🌟) ఇవ్వడం](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) మరియు [ఫోర్క్ చేయడం](https://github.com/microsoft/ai-agents-for-beginners/fork) మర్చిపోయవద్దు, తద్వారా కోడ్ నడిపించవచ్చు.

### ఇతర నేర్చుకునేవారిని కలుసుకోండి, మీ ప్రశ్నలకు సమాధానం పొందండి

మీరు ఏదైనా ఇబ్బంది పడితే లేదా AI ఏజెంట్లు నిర్మాణం గురించి ఏవైనా ప్రశ్నలు ఉంటే, మా ప్రత్యేక Discord ఛానెల్ లో చేరండి [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### మీకు అవసరం

ఈ కోర్సులో ప్రతి పాఠంలో కోడ్ ఉదాహరణలు ఉంటాయి, అవి code_samples ఫోల్డర్‌లో ఉన్నాయి. మీరు [ఈ రిపోజిని ఫోర్క్](https://github.com/microsoft/ai-agents-for-beginners/fork) చేసి మీ స్వంత ప్రతిని తయారు చేసుకోవచ్చు.

ఈ వ్యాయామాలలోని కోడ్ ఉదాహరణలు Microsoft Agent Framework ని Azure AI Foundry Agent Service V2 తో ఉపయోగిస్తాయి:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure అకౌంట్ అవసరం

ఈ కోర్స్ Microsoft నుండి ఈ క్రింది AI ఏజెంట్ ఫ్రేమ్‌వర్క్‌లను మరియు సేవలను ఉపయోగిస్తుంది:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


ఈ కోర్సు కోడ్ నడిపించుకానుకూడా మరిన్ని వివరాలకు [Course Setup](./00-course-setup/README.md) చూడండి.
