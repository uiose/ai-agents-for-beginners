##ఈవెంట్ పేరు: LangGraph మరియు Azure Cosmos DB తో బహుళ-ఏజెంట్ API

## వివరణ

బహుళ-ఏజెంట్ AI అప్లికేషన్ల వృద్ధి మేం ఎలా బుద్ధివంతమైన వ్యవస్థలను నిర్మిస్తామో మార్చేస్తోంది — కానీ మీరు వాటిని నిజమైన వరల్డ్ స్కేల్బిల్ మరియు పనితనానికి ఎలా ఆర్కిటెక్ట్ చేస్తారు? ఈ సెషన్లో, agent orchestration కోసం LangGraph, API పొర కోసం FastAPI, మరియు స్టేట్ మేనేజ్మెంట్, వెక్టర్ స్టోరేజ్ మరియు లావాదేవీ డేటాకు బ్యాక్బోన్‌గా Azure Cosmos DB ఉపయోగించి నిర్మించిన ప్రొడక్షన్-గ్రేడ్ బహుళ-ఏజెంట్ అప్లికేషన్‌లో ఒక లోతైన పర్యావలనం తీసుకుపోతాము.

వివరమైన కోడ్ వాక్‌థ్రూ ద్వారా, మీరు retrieval-augmented generation (RAG), మెమరీ పర్సిస్టెన్స్, మరియు డైనమిక్ స్టేట్ ట్రాన్సిషన్స్‌ను సజావుగా ఏకీకృతం చేసే ఏజెంట్-చలిత వర్క్‌ఫ్లోని ఎలా డిజైన్ చేసి అమలు చేయాలో చూస్తారు. మేము కవర్ చేస్తాము:

Agent collaboration with LangGraph for structured reasoning
Real-time chat history storage using Azure Cosmos DB - the same database that powers the chat history in ChatGPT, the fastest-growing AI agent application in history
Vector search for knowledge retrieval with Cosmos DB's native embeddings support
FastAPI’s async capabilities to keep interactions responsive and scalable

ఈ సెషన్ ముగిసే సమయానికి, ఆధునిక AI మరియు క్లౌడ్ ఇన్‌ఫ్రాస్ట్రక్చర్ శక్తిని వినియోగించే మీ స్వంత స్కేలబుల్, క్లౌడ్-నేటివ్ బహుళ-ఏజెంట్ అప్లికేషన్లను నిర్మించి డిప్లాయ్ చేయడానికి ఒక స్పష్టమైన బ్లూప్రింట్ మీ దగ్గర ఉంటుంది. మీరు AI ఇంజనీర్, క్లౌడ్ ఆర్కిటెక్ట్, లేదా Python డెవలపర్ అయితే, ఈ టాక్ మీకు ప్రాక్టికల్ అవగాహనలు మరియు బటిల్-టెస్టెడ్ ప్యాటర్న్స్‌ను అందిస్తుంది తద్వారా తదుపరి తరం AI-చలిత అనువర్తనాలను నిర్మించగలుగుతారు

## వివరణ

ఈ సెషన్లో మేము Microsoft Foundry ని ఉపయోగించి మీ Java అప్లికేషన్లలో అధునాతన AI ఏజెంట్ సామర్థ్యాలను ఎలా ఎంబెడ్ చేయాలో చూపిస్తాము, ఇందులో ప్రాజెక్ట్ లక్ష్యాలను సెట్ చేయడం, మోడల్స్‌తో ప్రయోగం చేయడం మరియు సురక్షితంగా ప్రొడక్షన్-సిద్ధ పరిష్కారాలను స్కేల్‌లో డిప్లాయ్ చేయడం অন্তর্ভুক্তం. మార్గం లో, GitHub Copilot (IntelliJ, VS Code, మరియు Eclipse లో) కోడింగ్ మరియు ప్రాంప్ట్ సృష్టిని ఎలా సులభతరం చేయగలదో మీకు తెలియజేస్తాము, అలాగే మోడల్ ఎంపిక, ఫైన్-ట్యూనింగ్, మరియు agentic వర్క్‌ఫ్లోల్లో ఉత్తమ ఆచారాలు బాధ్యతాయుతమైన మరియు సమర్ధవంతమైన అభివృద్ధిని నిర్ధారిస్తాయ్. మీరు AI ఏజెంట్లకు కొత్తవనైనా లేక అధునాతన ఏజెంట్-నిర్మాణ తంత్రాలపై ఆసక్తి ఉన్నవనైనా, ఈ సెషన్ మీరు ఇప్పటికే తెలుసుకుంటున్న టూల్స్ తో అంచనాల కన్నా మెరుగు అనుభవాలను అందించడానికి సామర్థ్యాన్ని ఇస్తుంది.

##ఈవెంట్ పేరు: LangChain4j మరియు డైనమిక్ సెషన్లను ఉపయోగించి Java AI ఏజెంట్లను నిర్మించడం

## వివరణ

LangChain4j ను Azure Container Apps (ACA) డైనమిక్ సెషన్లతో కలపడం ద్వారా మీ Java అప్లికేషన్లలో AI ఏజెంట్ల యొక్క సామర్థ్యాన్ని కల్తీ చేయండి, ఇవి Azure AI సేవలకు కనెక్ట్ అవుతాయి. ఈ సెషన్ ఒక ప్రాక్టికల్ ఉదాహరణను ప్రదర్శిస్తుంది, ఇందులో రిమోట్ వాతావరణంతో ఇంటరాక్ట్ చేయగల ఏజెంట్ నిర్మించబడినది, ఫైల్ నిర్వహణ సహా. కస్టమ్ టూల్స్‌ను నిర్వచించి, అవన్నిఆ ఏజెంట్ వర్క్‌ఫ్లోల్లో ఎలా సమీకరించాలో నేర్చుకోండి, మరియు బుద్ధిసమ్మత, డైనమిక్ పరిష్కారాలను డిప్లాయ్ చేయడానికి Azure యొక్క స్కేలబుల్ ఇన్‌ఫ్రాస్ట్రక్చర్‌ను ఎలా ఉపయోగించాలో తెలుసుకోండి.

##ఈవెంట్ పేరు: నిర్లక్ష్యమైన AI ఏజెంట్లు

## వివరణ

AI ఏజెంట్లలోని సంభావ్య రిస్క్‌లను పరిశీలిస్తూ, conversational AI ప్లాట్‌ఫారమ్‌లలో నమ్మకమైన AI ఆచారాలను ఎంబెడ్ చేయడం అనే సవాల్‌ను ఎలా ఎదుర్కోవాలో తలపోసుకుందాం! ఈ సెషన్ నిర్లక్ష్యమైన AI ఏజెంట్ల ఉదాహరణల్లో లోతుగా వెళ్తుంది — మోడల్ విఫలముల షాక్ చేసే ఉదాహరణలు, ప్రతికూల jailbreakలు, మరియు నమ్మకాన్ని మరియు అనుకూలాలను విప్పి వేయగలిగే ఇతర రిస్క్‌లను చూపిస్తుంది.

మేము Microsoft యొక్క నమ్మకమైన AI కోసం ఆధునిక టూల్స్‌ను పరిశీలిస్తాము, ఇందులో కంటెంట్ ఫిల్టర్లు, రెడ్-టీమింగ్ వ్యూహాలు, మరియు మూల్యాంకనాలు ఉన్నాయి — బాధ్యతాయుతంగా మరియు నిర్లక్ష్యంగా ప్రవర్తించే AI ఏజెంట్ల లైవ్ డెమోలను కూడా చూపిస్తూ.

🔥 మీరు తీసుకువెళ్ళబోయేదేమిటంటే:
✅ ఎక్స్‌ప్లోయిట్ కానుండే ముందు AI రిస్క్‌లను ఎలా గుర్తించి తగ్గించాలో
✅ Azure AI Content Safety ను ఉపయోగించి ప్రమాదకర ప్రవర్తనను ఎలా గుర్తించి తగ్గించాలో
✅ AI ఏజెంట్లను నమ్మకమైనవిగా చేయడానికి రహస్య సూత్రం

హైప్, అధిక-పందెం AI డ్రామా, మరియు మీ AIను చరిత్ర యొక్క సరైన వైపు ఉంచే తప్పనిసరి వ్యూహాలతో నిండిన సెషన్‌కు సిద్ధపడండి. కేవలం AI ను బిల్డ్ చేయకండి—ప్రసక్తిని కలిగించే AI ని నిర్మించండి!

##ఈవెంట్ పేరు: మీ కోడ్-ఫస్ట్ యాప్‌ను Azure AI Agent Service తో నిర్మించండి (.NET)

## వివరణ

Azure AI Agent Service సేవ మరియు SDK యొక్క సమగ్ర ఏకీకరణ, ఇది బలమైన AI-ఆధారిత పరిష్కాలాల అభివృద్ధిని సులభతరం చేస్తుంది. ఈ సెషన్లో, మీరు Azure మరియు C# ఉపయోగించి మీ స్వంత కోడ్-ఫస్ట్ AI ఏజెంట్‌ను ఎలా నిర్మించాలో నేర్చుకుంటారు — ఇది ప్రశ్నలకు సమాధానాలు ఇవ్వగలదు, డేటా విశ్లేషణ చేయగలదు, మరియు బాహ్య డేటా మూలాలను సమగ్రపరచగలదు. మీరు బహుళ ఏజెంట్లు కలిసి పనిచేసే వంటి మరింత క్లిష్టమైన ఆర్కిటెక్చర్లను కూడా పరిశీలిస్తారు.
