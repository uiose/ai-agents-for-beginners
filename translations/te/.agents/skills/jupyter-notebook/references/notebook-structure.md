# నోట్‌బుక్ నిర్మాణం

Jupyter నోట్‌బుక్స్ ఈ ఉండే ఉన్నత స్థాయి ఆకారంలో ఉన్న JSON డాక్యుమెంట్లు:

- `nbformat` మరియు `nbformat_minor`
- `metadata`
- `cells` (markdown మరియు కోడ్ సెల్స్ యొక్క జాబితా)

ప్రోగ్రామాటిక్గా `.ipynb` ఫైళ్ళను సవరించే సమయంలో:

- టెంప్లేట్‌లోని `nbformat` మరియు `nbformat_minor` ను నిలుపుకోండి.
- `cells` ను క్రమబద్ధమైన జాబితాగా ఉంచండి; ఉద్దేశ్యపూర్వకంగా కాకుంటే వాటి క్రమాన్ని మార్చవద్దు.
- కోడ్ సెల్స్ కోసం, తెలియకపోతే `execution_count` ను `null` గా పెట్టండి.
- కోడ్ సెల్స్ కోసం, scaffolding చేస్తున్నప్పుడు `outputs` ని ఖాళీ జాబితాగా పెట్టండి.
- మార్క్‌డౌన్ సెల్స్ కోసం, `cell_type="markdown"` మరియు `metadata={}` ను అలాగే ఉంచండి.

బండిల్ చేయబడిన టెంప్లేట్స్ లేదా `new_notebook.py` (ఉదాహరణకు, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) నుండి scaffolding చేయడాన్ని ప్రాధాన్యం ఇవ్వండి, బదులుగా రా నోట్‌బుక్ JSON ని చేతితో రాయకండి.
