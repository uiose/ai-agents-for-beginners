# AGENTS.md

## ప్రాజెక్ట్ అవలోకనం

ఈ రిపోజిటరీ "AI Agents for Beginners" ను కలిగి ఉంది - AI ఏజెంట్లు నిర్మించడానికి అవసరమైన అన్ని విషయాలను బోధించే సమగ్ర విద్యా కోర్సు. ఈ కోర్సు 15+ పాఠాలను కలిగి ఉంది, ఇందులో మూలాలు, డిజైన్ నమూనాలు, ఫ్రేమ్‌వర్క్లు మరియు AI ఏజెంట్ల ఉత్పత్తి ఉపయోగానికి సంబంధించిన అంశాలు ఉంటాయి.

**ప్రధాన సాంకేతికతలు:**
- Python 3.12+
- సంభాషణాత్మక尺ెర్చుకోల కోసం Jupyter నోట్‌బుక్స్
- AI ఫ్రేమ్‌వర్క్లు: Microsoft Agent Framework (MAF)
- Azure AI సేవలు: Microsoft Foundry, Azure AI Foundry Agent Service V2

**ఆర్కిటెక్చర్:**
- పాఠాల ఆధారిత నిర్మాణం (00-15+ డైరెక్టరీలు)
- ప్రతి పాఠం కలిగి ఉంటుంది: README డాక్యుమెంటేషన్, కోడ్ నమూనాలు (Jupyter నోట్‌బుక్స్), మరియు చిత్రాలు
- ఆటోమేటెడ్ అనువాద వ్యవస్థ ద్వారా బహుభాషా మద్దతు
- Microsoft Agent Framework ఉపయోగించే ప్రతి పాఠానికి ఒక Python నోట్‌బుక్

## సెటప్ ఆదేశాలు

### ముందస్తు అవసరాలు
- Python 3.12 లేదా అంతకంత ఎక్కువ
- Azure సబ్‌స్క్రిప్షన్ (Azure AI Foundry కోసం)
- Azure CLI సంస్థాపించి ధృవీకరించబడినది (`az login`)

### ప్రారంభ సెటప్

1. **రిపోజిటరీని క్లోన్ లేదా ఫోర్క్ చేయండి:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # లేదా
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python వర్చువల్ ఎన్విరాన్స్ సృష్టించి సక్రియం చేయండి:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # విండోస్‌పై: venv\Scripts\activate
   ```

3. **అనువర్తనాలను ఇన్‌స్టాల్ చేయండి:**
   ```bash
   pip install -r requirements.txt
   ```

4. **ఎన్విరాన్స్ వేరియబుల్స్ సెటప్ చేయండి:**
   ```bash
   cp .env.example .env
   # మీ API కీలు మరియు ఎండ్పాయింట్లతో .env ను సవరించండి
   ```

### అవసరమైన ఎన్విరాన్స్ వేరియబుల్స్

**Azure AI Foundry** (అవసరం):
- `AZURE_AI_PROJECT_ENDPOINT` - Azure AI Foundry ప్రాజెక్ట్ ఎండ్పాయింట్
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - మోడల్ డిప్లాయ్‌మెంట్ పేరు (ఉదాహరణకు, gpt-4o)

**Azure AI Search** (పాఠం 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search ఎండ్పాయింట్
- `AZURE_SEARCH_API_KEY` - Azure AI Search API కీ

ధ్రువీకరణ: నోట్‌బుక్స్ అమలు చేసే ముందు `az login` ను నిర్వహించండి (`AzureCliCredential` ఉపయోగిస్తుంది).
