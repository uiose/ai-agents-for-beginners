# AGENTS.md

## Project Overview

এই রেপোজিটরি "শুরুদের জন্য AI এজেন্টস" ধারণ করেছে - একটি বিস্তৃত শিক্ষামূলক কোর্স যা AI এজেন্ট তৈরি করার জন্য প্রয়োজনীয় সবকিছু শেখায়। কোর্সে ১৫+ পাঠ রয়েছে যা ফান্ডামেন্টালস, ডিজাইন প্যাটার্ন, ফ্রেমওয়ার্ক এবং AI এজেন্টের প্রোডাকশন ডিপ্লয়মেন্ট কভার করে।

**মূল প্রযুক্তিসমূহ:**
- পাইথন ৩.১২+
- ইন্টারেক্টিভ শিক্ষার জন্য জুপিটার নোটবুক
- AI ফ্রেমওয়ার্ক: Microsoft Agent Framework (MAF)
- Azure AI সার্ভিস: Microsoft Foundry, Azure AI Foundry Agent Service V2

**আর্কিটেকচার:**
- লেসন ভিত্তিক গঠন (০০-১৫+ ডিরেক্টরিস)
- প্রতিটি লেসনে রয়েছে: README ডকুমেন্টেশন, কোড স্যাম্পল (জুপিটার নোটবুক), এবং ইমেজ
- স্বয়ংক্রিয় অনুবাদ সিস্টেমের মাধ্যমে বহু-ভাষার সমর্থন
- Microsoft Agent Framework ব্যবহার করে প্রতি লেসনে একটি পাইথন নোটবুক

## Setup Commands

### প্রাক-শর্ত
- পাইথন ৩.১২ বা তার উপরে
- Azure সাবস্ক্রিপশন (Azure AI Foundry জন্য)
- Azure CLI ইনস্টল এবং প্রমাণীকৃত (`az login`)

### প্রাথমিক সেটআপ

১. **রেপোজিটরি ক্লোন বা ফর্ক করুন:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # অথবা
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

২. **Python ভার্চুয়াল এনভায়রনমেন্ট তৈরি ও সক্রিয় করুন:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # উইন্ডোজে: venv\Scripts\activate
   ```

৩. **ডিপেন্ডেন্সি ইনস্টল করুন:**
   ```bash
   pip install -r requirements.txt
   ```

৪. **এনভায়রনমেন্ট ভ্যারিয়েবল সেট করুন:**
   ```bash
   cp .env.example .env
   # আপনার API কী এবং এন্ডপয়েন্ট সহ .env সম্পাদনা করুন
   ```

### প্রয়োজনীয় এনভায়রনমেন্ট ভ্যারিয়েবলসমূহ

**Azure AI Foundry** (আবশ্যক):
- `AZURE_AI_PROJECT_ENDPOINT` - Azure AI Foundry প্রকল্পের এন্ডপয়েন্ট
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - মডেল ডিপ্লয়মেন্টের নাম (উদাহরণ: gpt-4o)

**Azure AI Search** (লেসন ০৫ - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search এর এন্ডপয়েন্ট
- `AZURE_SEARCH_API_KEY` - Azure AI Search এর API কী

প্রমাণীকরণ: নোটবুক চালানোর আগে `az login` চালান (ব্যবহার করে `AzureCliCredential`)।

## Development Workflow

### জুপিটার নোটবুক চালানো

প্রতিটি লেসনে বিভিন্ন ফ্রেমওয়ার্কের জন্য একাধিক জুপিটার নোটবুক রয়েছে:

১. **জুপিটার শুরু করুন:**
   ```bash
   jupyter notebook
   ```

২. **একটি লেসন ডিরেক্টরিতে নেভিগেট করুন** (যেমন, `01-intro-to-ai-agents/code_samples/`)

৩. **নোটবুক খুলুন এবং চালান:**
   - `*-python-agent-framework.ipynb` - Microsoft Agent Framework (পাইথন) ব্যবহার করে
   - `*-dotnet-agent-framework.ipynb` - Microsoft Agent Framework (.NET) ব্যবহার করে

### Microsoft Agent Framework এর সাথে কাজ করা

**Microsoft Agent Framework + Azure AI Foundry:**
- Azure সাবস্ক্রিপশন প্রয়োজন
- Agent Service V2 জন্য `AzureAIProjectAgentProvider` ব্যবহার করে (এজেন্টরা Foundry পোর্টালে দৃশ্যমান)
- প্রোডাকশন রেডি বিল্ট-ইন অবজার্ভেবিলিটির সাথে
- ফাইল প্যাটার্ন: `*-python-agent-framework.ipynb`

## Code Style

### পাইথন প্রবণতা

- **পাইথন ভার্সন**: ৩.১২+
- **কোড স্টাইল**: স্ট্যান্ডার্ড পাইথন PEP 8 নিয়ম অনুসরণ করুন
- **নোটবুক**: ধারাবাহিক স্পষ্ট মার্কডাউন সেল ব্যবহার করুন ধারণা ব্যাখ্যার জন্য
- **ইমপোর্ট**: স্ট্যান্ডার্ড লাইব্রেরি, থার্ড পার্টি, লোকাল ইমপোর্ট আলাদা গ্রুপে রাখুন

### জুপিটার নোটবুক প্রবণতা

- কোড সেলের আগে বর্ণনামূলক মার্কডাউন সেল রাখুন
- রেফারেন্সের জন্য আউটপুট উদাহরণ নোটবুকে যুক্ত করুন
- লেসন ধারণার সাথে মিল রেখে স্পষ্ট ভ্যারিয়েবল নাম দিন
- নোটবুকের কার্যকরী ক্রম রেখেছেন লিনিয়ার (সেল ১ → ২ → ৩...)
