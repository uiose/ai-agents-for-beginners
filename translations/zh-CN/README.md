## 🌱 快速开始

本课程涵盖构建AI代理的基础知识。每节课都有独立主题，您可以随意开始！

本课程支持多语言。详见我们的[可用语言](../..)。

如果您是首次使用生成式AI模型构建，推荐查看我们的[生成式AI初学者课程](https://aka.ms/genai-beginners)，包含21节关于生成式AI构建的课程。

别忘了[给本仓库加星（🌟）](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst)并[派生本仓库](https://github.com/microsoft/ai-agents-for-beginners/fork)来运行代码。

### 认识其他学习者，获取你的问题答案

如果遇到困难或有关于构建AI代理的任何疑问，欢迎加入我们在[Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)中的专属Discord频道。

### 你需要准备的

本课程每节课都包括代码示例，存放在code_samples文件夹中。您可以[派生本仓库](https://github.com/microsoft/ai-agents-for-beginners/fork)创建自己的副本。

这些示例代码使用微软代理框架和Azure AI Foundry Agent Service V2：

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - 需要Azure账户

本课程使用了微软以下AI代理框架与服务：

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

关于本课程代码运行的更多信息，请参阅[课程设置](./00-course-setup/README.md)。
