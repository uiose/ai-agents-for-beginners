>
> Dette repoet inneholder 50+ oversettelser som øker nedlastingsstørrelsen betydelig. For å klone uten oversettelser, bruk sparse checkout:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Dette gir deg alt du trenger for å fullføre kurset med en mye raskere nedlasting.

## 🌱 Kom i gang

Dette kurset inneholder leksjoner som dekker grunnprinsippene for å bygge AI-agenter. Hver leksjon tar for seg sitt eget emne, så start hvor du vil!

Dette kurset har flerspråklig støtte. Gå til våre [tilgjengelige språk her](../..). 

Hvis dette er første gang du bygger med generative AI-modeller, sjekk ut vårt [Generative AI For Beginners](https://aka.ms/genai-beginners) kurs, som inkluderer 21 leksjoner om bygging med GenAI.

Ikke glem å [gi stjerne (🌟) til dette repoet](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) og [forke dette repoet](https://github.com/microsoft/ai-agents-for-beginners/fork) for å kjøre koden.

### Møt andre deltakere, få svar på spørsmålene dine

Hvis du står fast eller har spørsmål om å bygge AI-agenter, bli med i vår dedikerte Discord-kanal i [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Hva du trenger

Hver leksjon i dette kurset inneholder kodeeksempler, som kan finnes i mappen code_samples. Du kan [forke dette repoet](https://github.com/microsoft/ai-agents-for-beginners/fork) for å lage din egen kopi.  

Kodeeksemplene i disse øvelsene benytter Microsoft Agent Framework med Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Krever Azure-konto

Dette kurset bruker følgende AI-agent-rammeverk og tjenester fra Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


For mer informasjon om å kjøre koden for dette kurset, gå til [Kursoppsett](./00-course-setup/README.md).
