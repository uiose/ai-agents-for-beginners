> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"

## 🌱 Începeți

Acest curs conține lecții care acoperă fundamentele construirii Agenților AI. Fiecare lecție acoperă propriul subiect, așa că începe de unde dorești!

Există suport multilingv pentru acest curs. Accesează limbile noastre disponibile [aici](../..). 

Dacă este pentru prima dată când construiești cu modele de Inteligență Generativă, consultă cursul nostru [Generative AI For Beginners](https://aka.ms/genai-beginners), care include 21 de lecții despre construirea cu GenAI.

Nu uita să [acordezi o stea (🌟) acestui repo](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) și să [faci fork la acest repo](https://github.com/microsoft/ai-agents-for-beginners/fork) pentru a rula codul.

### Întâlnește alți cursanți, primește răspunsuri la întrebările tale

Dacă te blochezi sau ai orice întrebări despre construirea Agenților AI, alătură-te canalului nostru dedicat de Discord în [Discord Microsoft Foundry](https://aka.ms/ai-agents/discord).

### Ce îți trebuie

Fiecare lecție din acest curs include exemple de cod, care pot fi găsite în folderul code_samples. Poți [fă un fork la acest repo](https://github.com/microsoft/ai-agents-for-beginners/fork) pentru a-ți crea propria copie.  

Exemplele de cod din aceste exerciții utilizează Microsoft Agent Framework cu Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Cont Azure necesar

Acest curs folosește următoarele cadre și servicii pentru Agenți AI de la Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Pentru mai multe informații despre rularea codului pentru acest curs, mergi la [Course Setup](./00-course-setup/README.md).
