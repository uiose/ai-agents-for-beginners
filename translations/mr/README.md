> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> यामुळे तुम्हाला कोर्स पूर्ण करण्यासाठी आवश्यक सर्वकाही मिळते आणि डाउनलोड खूप वेगवान होते.

## 🌱 सुरूवात

हा कोर्स AI एजंट्स तयार करण्याच्या मूलभूत गोष्टींवर आधारित धडे देतो. प्रत्येक धडा स्वतःचा विषय समाविष्ट करतो, त्यामुळे तुम्हाला जिथे आवडेल तिथे सुरू करा!

या कोर्ससाठी बहुभाषिक समर्थन उपलब्ध आहे. आमच्या [उपलब्ध भाषा इथे पहा](../..). 

जर हे जनरेटिव्ह AI मॉडेल्ससह तुमचे पहिले प्रकल्प असेल, तर आमचा [Generative AI For Beginners](https://aka.ms/genai-beginners) कोर्स पहा, ज्यात GenAI वापरून बनवण्यावर 21 धडे आहेत.

या रेपोला [स्टार (🌟) द्या](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) आणि कोड चालवण्यासाठी [हा रेपो फोर्क करा](https://github.com/microsoft/ai-agents-for-beginners/fork) हे विसरू नका.

### इतर शिकणाऱ्यांना भेटा, तुमचे प्रश्न विचारून उत्तर मिळवा

जर तुम्ही अडकलात किंवा AI एजंट्स तयार करण्यासंबंधी काही प्रश्न असतील तर [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) मधील आमच्या समर्पित Discord चॅनेलमध्ये सहभागी व्हा.

### काय आवश्यक आहे

या कोर्समधील प्रत्येक धड्यात कोड उदाहरणे आहेत, जी code_samples फोल्डरमध्ये सापडतील. आपली प्रत तयार करण्यासाठी आपण हा रेपो [फोर्क करू शकता](https://github.com/microsoft/ai-agents-for-beginners/fork).  

या व्यायामांमधील कोड उदाहरणे Microsoft Agent Framework सह Azure AI Foundry Agent Service V2 वापरतात:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure खाते आवश्यक

हा कोर्स Microsoft कडून खालील AI एजंट फ्रेमवर्क आणि सेवा वापरतो:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


कोर्ससाठी कोड चालवण्याबद्दल अधिक माहितीसाठी, [Course Setup](./00-course-setup/README.md) पहा.
