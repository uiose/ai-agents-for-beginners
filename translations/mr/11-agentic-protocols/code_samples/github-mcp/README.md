# Github MCP सर्व्हर उदाहरण

## वर्णन

हा Microsoft Reactor द्वारे आयोजित AI एजंट्स हॅकाथॉनसाठी तयार केलेला डेमो आहे.

हे टूल वापरकर्त्याच्या Github रेपोजवर आधारित हॅकाथॉन प्रकल्पांची शिफारस करण्यासाठी वापरले जाते.
हे खालील प्रमाणे केले जाते:

1. **Github एजंट** - Github MCP सर्व्हर वापरून रेपोज आणि त्या रेपोजबाबतची माहिती मिळवणे.
2. **Hackathon एजंट** - Github एजंटकडून मिळालेली माहिती घेऊन प्रकल्प, वापरकर्त्याद्वारे वापरलेल्या भाषां आणि AI एजंट्स हॅकाथॉनसाठी प्रकल्प ट्रॅकवर आधारित सर्जनशील हॅकाथॉन प्रकल्पांच्या कल्पना सुचवणे.
3. **इव्हेंट्स एजंट** - हॅकाथॉन एजंटच्या सूचनांवरून, इव्हेंट्स एजंट AI एजंट हॅकाथॉन सिरीजमधील संबंधित इव्हेंट्सची शिफारस करेल.

## कोड चालविणे 

### पर्यावरण बदल

हा डेमो Microsoft Agent Framework, Azure OpenAI सेवा, Github MCP सर्व्हर आणि Azure AI Search वापरतो.

हे टूल वापरण्यासाठी खात्री करा की योग्य पर्यावरण बदल सेट केले आहेत:

```python
AZURE_AI_PROJECT_ENDPOINT=""
AZURE_AI_MODEL_DEPLOYMENT_NAME=""
AZURE_SEARCH_SERVICE_ENDPOINT=""
AZURE_SEARCH_API_KEY=""
``` 

## Chainlit सर्व्हर चालू करणे

MCP सर्व्हरशी कनेक्ट करण्यासाठी, हा डेमो Chainlit चा चॅट इंटरफेस म्हणून वापरतो.

सर्व्हर चालू करण्यासाठी, आपल्या टर्मिनलमध्ये खालील कमांड वापरा:

```bash
chainlit run app.py -w
```

यामुळे आपला Chainlit सर्व्हर `localhost:8000` वर सुरू होईल तसेच `event-descriptions.md` चा कंटेंट आपल्या Azure AI Search इंडेक्समध्ये भरले जाईल.

## MCP सर्व्हरशी कनेक्ट करणे

Github MCP सर्व्हरशी कनेक्ट होण्यासाठी, "Type your message here.." च्या खाली असलेल्या "प्लग" आयकॉनवर क्लिक करा:

![MCP Connect](../../../../../translated_images/mr/mcp-chainlit-1.7ed66d648e3cfb28.webp)

तिथून आपण "Connect an MCP" वर क्लिक करून Github MCP सर्व्हरशी कनेक्ट होण्यासाठी कमांड जोडू शकता:

```bash
npx -y @modelcontextprotocol/server-github --env GITHUB_PERSONAL_ACCESS_TOKEN=[YOUR PERSONAL ACCESS TOKEN]
```

"[YOUR PERSONAL ACCESS TOKEN]" हे आपल्या प्रत्यक्ष वैयक्तिक प्रवेश टोकनने बदला.

कनेक्ट केल्यानंतर, प्लग आयकॉनजवळ (1) दिसेल जे कनेक्ट असल्याचे संकेत देते. न दिसल्यास, `chainlit run app.py -w` वापरून chainlit सर्व्हर रीस्टार्ट करण्याचा प्रयत्न करा.

## डेमो वापरणे

हॅकाथॉन प्रकल्पांची शिफारस करण्यासाठी एजंट कार्यप्रवाह सुरू करण्यासाठी, तुम्ही खालीलप्रमाणे संदेश टाइप करू शकता:

"Recommend hackathon projects for the Github user koreyspace"

Router Agent तुमची विनंती विश्लेषण करून कोणत्या एजंट्सच्या (GitHub, Hackathon, आणि Events) संयोजनाने तुमच्या प्रश्नाचे उत्तम उत्तर देता येईल हे ठरवेल. एजंट्स एकत्र काम करून GitHub रेपॉझिटरी विश्लेषण, प्रकल्प कल्पना, आणि संबंधित तांत्रिक कार्यक्रमांवर आधारित सर्वसमावेशक शिफारसी देतात.
