## 🌱 ಪ್ರಾರಂಭಿಸುವುದು

ಈ ಕೋರ್ಸ್ AI ಏಜೆಂಟ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವ ಮೂಲಗಳನ್ನು ಒಳಗೊಂಡ ಪಾಠಗಳನ್ನು ಹೊಂದಿದೆ. ಪ್ರತಿ ಪಾಠವು ತನ್ನದೇ ವಿಚಾರವನ್ನು ಒಳಗೊಂಡಿದ್ದು ನೀವು ಇಚ್ಛಿಸಿದ ಎಲ್ಲಿಯೂ ಪ್ರಾರಂಭಿಸಬಹುದು!

ಈ ಕೋರ್ಸ್‌ಗೆ ಬಹು-ಭಾಷಾ ಬೆಂಬಲವಿದೆ. ನಮ್ಮ [ಲಭ್ಯವಿರುವ ಭಾಷೆಗಳ ಪಟ್ಟಿ](../..)ಗೆ ಹೋಗಿ.

ನೀವು ಜನರೇಟಿವ್ AI ಮಾದರಿಗಳಲ್ಲಿ ಮೊದಲ ಬಾರಿಗೆ ನಿರ್ಮಿಸುತ್ತಿದ್ದರೆ, ನಮ್ಮ [ಅಾರಂಭಿಕರಿಗಾಗಿ ಜನರೇಟಿವ್ AI](https://aka.ms/genai-beginners) ಕೋರ್ಸ್ ನೋಡಬಹುದು, ಇದರಲ್ಲಿ 21 ಪಾಠಗಳಿವೆ.

ಈ ರೆಪೊಗೆ [ನಕ್ಷತ್ರ (🌟) ನೀಡಿ](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ಮತ್ತು [ಈ ರೆಪೊ ಫೋರ್ಕ್ ಮಾಡಿ](https://github.com/microsoft/ai-agents-for-beginners/fork) ಕೋಡ್ ಚಲಾಯಿಸಲು ಮರೆತಿಬೇಡಿ.

### ಇತರ ಶಿಕ್ಷಣಾರ್ಥಿಗಳನ್ನು ಭೇಟಿ ಮಾಡಿ, ನಿಮ್ಮ ಪ್ರಶ್ನೆಗಳಿಗೆ ಉತ್ತರ ಪಡೆಯಿರಿ

ನೀವು ಹಿಂದೆ ಸಿಲುಕಿದಲ್ಲಿ ಅಥವಾ AI ಏಜೆಂಟ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವ ಕುರಿತು ಯಾವುದೇ ಪ್ರಶ್ನೆಗಳಿದ್ದರೆ, ನಮ್ಮ ವಿಶೇಷ Discord ಚಾನಲ್‌ನಲ್ಲಿ ಸೇರಿ [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) ನಲ್ಲಿ.

### ನಿಮಗೆ ಬೇಕಾಗಿರುವವು

ಈ ಕೋರ್ಸ್‌ನ ಪ್ರತಿಯೊಂದು ಪಾಠವೂ ಕಡತ ಉದಾಹರಣೆಗಳನ್ನು ಒಳಗೊಂಡಿದೆ, ಇನ್ನೂ ತಿಳಿದುಕೊಳ್ಳಲು [ಕೋಡ್_ಸ್ಯಾಂಪಲ್ಸ್ ಫೋಲ್ಡರ್‌ನಲ್ಲಿ](https://github.com/microsoft/ai-agents-for-beginners/fork) ಈ ರೆಪೊವನ್ನು ಫೋರ್ಕ್ ಮಾಡಿ ನಿಮ್ಮ ಸ್ವಂತ ನಕಲನ್ನು ರಚಿಸಬಹುದು.

ಈ ವ್ಯಾಯಾಮಗಳಲ್ಲಿನ ಕೋಡ್ ಉದಾಹರಣೆಗಳು ಮೈಸೂರ್‌ರಾಫ್ಟ್ ಏಜೆಂಟ್ ಫ್ರೆ임್‌ವರ್ಕ್ ಮತ್ತು ಅಜುರ್ AI ಫೌಂಡ್ರಿ ಏಜೆಂಟ್ ಸರ್ವಿಸ್ V2 ಬಳಸುತ್ತವೆ:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - ಅಜುರ್ ಖಾತೆ ಅಗತ್ಯ

ಈ ಕೋರ್ಸ್ Microsoft ನಿಂದ ಕೆಳಗಿನ AI ಏಜೆಂಟ್ ಫ್ರೆ임್‌ವರ್ಕ್ ಮತ್ತು ಸೇವೆಗಳನ್ನು ಬಳಸುತ್ತದೆ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


ಈ ಕೋರ್ಸ್‌ನ ಕೋಡ್ ಚಾಲನೆಯ ಕುರಿತ ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ [ಕೋರ್ಸ್ ಸೆಟ್‌ಅಪ್](./00-course-setup/README.md) ಗೆ ಹೋದಿರಿ.
