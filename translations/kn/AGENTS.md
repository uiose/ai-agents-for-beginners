# AGENTS.md

## Project Overview

ಈ ರೆಪೊಸಿಟರಿ "ಆಯ್‌ಐ ಏಜೆಂಟ್ಸ್ ಫಾರ್ ಬೆಗಿನ್ನರ್ಸ್" ಅನ್ನು ಒಳಗೊಂಡಿದೆ - ಎಐ ಏಜೆಂಟ್ಸ್ ನಿರ್ಮಿಸಲು ಬೇಕಾಗುವ ಎಲ್ಲಾ ವಿಷಯಗಳನ್ನು ಕಲಿಸುವ ಆವರ್ತಿತ ಶಿಕ್ಷಣ ಕೋರ್ಸ್. ಕೋರ್ಸ್ 15+ ಪಾಠಗಳನ್ನು ಒಳಗೊಂಡಿದ್ದು, ಮೂಲಭೂತ ತತ್ವಗಳು, ವಿನ್ಯಾಸ ಮಾದರಿಗಳು, ಫ್ರೇಮ್ವರ್ಕ್‌ಗಳು ಮತ್ತು ಎಐ ಏಜೆಂಟ್ಸ್‌ಗಳ ಉತ್ಪಾದನಾ ನಿಯೋಜನೆಯನ್ನು ಒಳಗೊಂಡಿದೆ.

**ಪ್ರಮುಖ ತಂತ್ರಜ್ಞಾನಗಳು:**
- Python 3.12+
- ಇಂಟರેક્ટಿವ್ ಕಲಿಕೆಗೆ Jupyter नोटಬುಕ್‌ಗಳು
- ಎಐ ಫ್ರೇಮ್ವರ್ಕ್‌ಗಳು: Microsoft Agent Framework (MAF)
- Azure AI ಸೇವೆಗಳು: Microsoft Foundry, Azure AI Foundry Agent Service V2

**ವಾಸ್ತುಶಿಲ್ಪ:**
- ಪಾಠ ಆಧಾರಿತ ರಚನೆ (00-15+ ಡೈರೆಕ್ಟರಿಗಳು)
- ಪ್ರತಿ ಪಾಠದಲ್ಲಿ ಅಂತರಾಳಿತ ದಾಖಲೆ, ಕೋಡ್ ಮಾದರಿಗಳು (Jupyter ನೋಟುಬುಕ್‌ಗಳು), ಮತ್ತು ಚಿತ್ರಗಳು ಇರುತ್ತವೆ
- ಸ್ವಯಂಚಾಲಿತ ಅನುವಾದ ವ್ಯವಸ್ಥೆಯ ಮೂಲಕ ಬಹುಭಾಷಾ ಬೆಂಬಲ
- ಪ್ರತಿ ಪಾಠಕ್ಕೆ ಒಬ್ಬ Python ನೋಟುಬುಕ್ Microsoft Agent Framework ಬಳಸಿ

## Setup Commands

### Prerequisites
- Python 3.12 ಅಥವಾ ಹೆಚ್ಚಿನ ಆವೃತ್ತಿ
- Azure ಸಬ್‌ಸ್ಕ್ರಿಪ್ಷನ್ (Azure AI Foundry ಗಾಗಿ)
- Azure CLI ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿಕೊಂಡು ಲಾಗಿನ್ ಆಗಿರಬೇಕು (`az login`)

### Initial Setup

1. **ರೆಪೊಸಿಟರಿ ಕ್ಲೋನ್ ಅಥವಾ ಫೋರ್ಕ್ ಮಾಡಿ:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # ಅಥವಾ
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python ವರ್ಚುವಲ್ ಎನ್‌ವೈರನ್ಮೆಂಟ್ ರಚಿಸಿ ಮತ್ತು ಸಕ್ರಿಯಗೊಳಿಸಿ:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # ವಿಂಡೋಸ್‌ನಲ್ಲಿ: venv\Scripts\activate
   ```

3. **ನಿರ್ವಹಣಾ ಪ್ಯಾಕೇಜುಗಳನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿ:**
   ```bash
   pip install -r requirements.txt
   ```

4. **ಪರಿಸರ ಚರಗಳನ್ನು ಸೆಟ್ ಮಾಡಿ:**
   ```bash
   cp .env.example .env
   # ನಿಮ್ಮ API ಕೀಲಿಗಳು ಮತ್ತು ಎಂಡ್‌ಪಾಯಿಂಟ್‌ಗಳೊಂದಿಗೆ .env ಅನ್ನು ಸಂಪಾದಿಸಿ
   ```

### Required Environment Variables

**Azure AI Foundry** (ಆವಶ್ಯಕ):
- `AZURE_AI_PROJECT_ENDPOINT` - Azure AI Foundry ಯೋಜನೆ ಅಂತಿಮ ಬಿಂದು
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - ಮಾದರಿ ನಿಯೋಜನೆ ಹೆಸರು (ಉದಾ: gpt-4o)

**Azure AI Search** (ಪಾಠ 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search ಅಂತಿಮ ಬಿಂದು
- `AZURE_SEARCH_API_KEY` - Azure AI Search API ಕೀ

ಪ್ರಮಾಣೀಕರಣ: ನೋಟುಬುಕ್‌ಗಳನ್ನು ಚಾಲನೆಯಲ್ಲಿಡುವ ಮೊದಲು `az login` ಮಾಡಿ (`AzureCliCredential` ಬಳಕೆ).

## Development Workflow

### Jupyter ನೋಟುಬುಕ್‌ಗಳು ಚಾಲನೆ

ಪ್ರತಿ ಪಾಠದಲ್ಲಿ ವಿವಿಧ ಫ್ರೇಮ್ವರ್ಕ್‌ಗಳಿಗಾಗಿ ಬಹು ನೋಟುಬುಕ್‌ಗಳು ಇರುತ್ತವೆ:

1. **Jupyter ಪ್ರಾರಂಭಿಸಿ:**
   ```bash
   jupyter notebook
   ```

2. **ಪಾಠ ಡೈರೆಕ್ಟರಿಗೆ ನಾವಿಗೇಟ್ ಮಾಡಿ** (ಉದಾಹರಣೆಗೆ `01-intro-to-ai-agents/code_samples/`)

   - `*-python-agent-framework.ipynb` - Microsoft Agent Framework (Python) ಬಳಸಿ
   - `*-dotnet-agent-framework.ipynb` - Microsoft Agent Framework (.NET) ಬಳಸಿ

### Microsoft Agent Framework ನೊಂದಿಗೆ ಕೆಲಸ

   - ಕೆಲವು ನೋಟುಬುಕ್‌ಗಳಿಗೆ ಅನುವಾಗಿ ವಿಳಂಬ ಪ್ಯಾಕೇಜುಗಳು markdown ನಲ್ಲಿ ಒದಗಿಸಬಲ್ಲವು
