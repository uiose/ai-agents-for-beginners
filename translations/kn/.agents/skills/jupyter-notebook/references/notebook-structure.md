# ನೋಟ್ಬುಕ್ ರಚನೆ

Jupyter ನೋಟ್ಬುಕ್‌ಗಳು ಈ ಉನ್ನತ-ಮಟ್ಟದ ಆಕಾರವನ್ನು ಹೊಂದಿರುವ JSON ದಸ್ತಾವೇಜುಗಳಾಗಿವೆ:

- `nbformat` ಮತ್ತು `nbformat_minor`
- `metadata`
- `cells` (markdown ಮತ್ತು code ಸೆಲ್‌ಗಳ ಪಟ್ಟಿಯು)

ಪ್ರೋಗ್ರಾಮ್ಯಾಟಿಕವಾಗಿ `.ipynb` ಫೈಲ್‌ಗಳನ್ನು ಸಂಪಾದಿಸುವಾಗ:

- ಟೆಂಪ್ಲೇಟಿನಿಂದ `nbformat` ಮತ್ತು `nbformat_minor` ಅನ್ನು ಉಳಿಸಿ.
- `cells` ಅನ್ನು ಕ್ರಮಬದ್ಧ ಪಟ್ಟಿಯಾಗಿ ಇಡಿ; ಉದ್ದೇಶಿತವಾಗದಿದ್ದರೆ ಮರುಕ್ರಮಗೊಳಿಸಬೇಡಿ.
- ಕೋಡ್ ಸೆಲ್‌ಗಳಿಗಾಗಿ, `execution_count` ಅನ್ನು ತಿಳಿಯದಿದ್ದರೆ `null` ಗೆ ಹೊಂದಿಸಿ.
- ಕೋಡ್ ಸೆಲ್‌ಗಳಿಗಾಗಿ, ಆರಂಭಿಕ ರಚನೆ ಮಾಡುವಾಗ `outputs` ಅನ್ನು ಖಾಲಿ ಪಟ್ಟಿಯಾಗಿ ಹೊಂದಿಸಿ.
- markdown ಸೆಲ್‌ಗಳಿಗಾಗಿ, `cell_type="markdown"` ಮತ್ತು `metadata={}` ಅನ್ನು ಉಳಿಸಿ.

ಕೈಯಿಂದ ಕಚ್ಚಾ ನೋಟ್ಬುಕ್ JSON ಅನ್ನು ರಚಿಸುವ ಬದಲು, ಪ್ಯಾಕ್ ಮಾಡಲಾದ ಟೆಂಪ್ಲೇಟುಗಳು ಅಥವಾ `new_notebook.py` (ಉದಾಹರಣೆಗೆ, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) ಮೂಲಕ ಆರಂಭಿಕ ರಚನೆಯನ್ನು ಪ್ರಾಥಮ್ಯ ನೀಡಿ.
