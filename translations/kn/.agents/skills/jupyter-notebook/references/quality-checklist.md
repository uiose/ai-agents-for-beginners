# ಗುಣಮಟ್ಟ ಪರಿಶೀಲನಾ ಪಟ್ಟಿ

ನೋಟ್‌ಬುಕ್ ವಿತರಿಸುವ ಮೊದಲು:

- ಅದನ್ನು ಮೇಲಿಂದ ಕೆಳಗೆ ಕನಿಷ್ಠ ಒಂದು ಬಾರಿ ಚಾಲನೆ ಮಾಡಿ (ಅಥವಾ ಪರಿಸರವು ಅನುಮತಿಸುವಷ್ಟರಷ್ಟೂ).
- ಪ್ರಾರಂಭಿಕ ಸೆಲ್‌ಗಳು ಎಲ್ಲ ಅಗತ್ಯ ಸ್ಥಿತಿಯನ್ನು ಹೊಂದಿಸುತ್ತವೆ ಎಂದು ಖಚಿತಪಡಿಸಿ; ಹಿಂದಿನ ರನ್‌ಗಳಿಂದ ಉಳಿದಿರುವ ಗುಪ್ತ ಸ್ಥಿತಿಯನ್ನು ತಪ್ಪಿಸಿ.
- ಔಟ್‌ಪುಟ್‌ಗಳನ್ನು ಶುಚಿಯಾಗಿರಿಸಿ. ಸಣ್ಣ ಸಾರಾಂಶ ಕೆಲಸ ಮಾಡುವಾಗ ಭಾರೀ ಔಟ್‌ಪುಟ್‌ಗಳನ್ನು ತಪ್ಪಿಸಿ.
- ಸಣ್ಣ ಟೇಬಲ್ಗಳು, ಪ್ರಮುಖ ಮೆಟ್ರಿಕ್‌ಗಳು, ಅಥವಾ ಸಂಕ್ಷಿಪ್ತ ಪ್ರಿಂಟ್‌ಔಟ್‌ಗಳನ್ನು ಆದ್ಯತೆಯಾಗಿ ಬಳಸಿರಿ.
- ಕಥನವನ್ನು ತ್ವರಿತವಾಗಿ ಸ್ಕಿಮ್ ಮಾಡಬಹುದಾಗಿರಲಿ. ಶೀರ್ಷಿಕೆಗಳನ್ನು ಮತ್ತು ಸಂಕ್ಷಿಪ್ತ ಬುಲೆಟ್‌ಗಳನ್ನು ಬಳಸಿ, ಹಾಗೂ ದೀರ್ಘ ಪ್ಯಾರಾಗ್ರಾಫ್‌ಗಳನ್ನು ತಪ್ಪಿಸಿ.
- ಅವಶ್ಯಕವಾದಾಗ ಮಾತ್ರ ಸಹಾಯಕ TODOಗಳನ್ನು ಬಿಟ್ಟುಹೋಗಿ, ಮತ್ತು ಅವುಗಳನ್ನು ಸ್ಪಷ್ಟವಾಗಿ ಲೇಬಲ್ ಮಾಡಿ.
- ರನ್ ಮಾಡುವುದು ಸಾಧ್ಯವಿಲ್ಲದಿದ್ದರೆ, ಅಪಾಯವನ್ನು ಗಮನಿಸಿ ಮತ್ತು ಸ್ಥಳೀಯವಾಗಿ ಅದನ್ನು ಹೇಗೆ ಪರಿಶೀಲಿಸಬೇಕು ಎಂಬುದನ್ನು ವಿವರಿಸಿ.
