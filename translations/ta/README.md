> **உள்ளூர் பிரதியை மாற்ற விரும்புகிறீர்களா?**
>
> இந்த தொகுப்பு 50+ மொழி மொழிபெயர்ப்புகள் உள்ளடக்கியுள்ளதாக உள்ளது, இது பதிவிறக்கம் அளவை மிகவும் அதிகரிக்கிறது. மொழிபெயர்ப்புகளின்றி பிரதியை மாற்ற sparse checkout ஐ பயன்படுத்துங்கள்:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> இது பாடநெறியை முடிக்க தேவையான அனைத்தையும் மிக விரைவான பதிவிறக்கத்துடன் தரும்.

## 🌱 தொடங்குவது எப்படி

இந்த பாடநெறி, AI முகவர்களை கட்டமைப்பதற்கான அடிப்படைகள் கவனத்தில் கொண்டு பாடங்களைக் கொண்டுள்ளது. ஒவ்வொரு பாடமும் தனிப்பட்ட தலைப்பை உள்ளடக்கியது ஆகவே உங்கள் விருப்பமான இடத்தில் இருந்து துவங்கவும்!

இந்த பாடநெறிக்கான பலமொழி ஆதரவு உள்ளது. எங்கள் [இங்கு கிடைக்கும் மொழிகளுக்குச் செல்லவும்](../..).

இந்த முதற்சாரியாக Generative AI மாதிரிகளுடன் கட்டமைக்கும்போது, எங்கள் [Generative AI தொடங்குவோருக்கான பாடநெறி](https://aka.ms/genai-beginners) மற்றும் இதில் GenAI மூலம் கட்டமைக்க 21 பாடங்கள் உள்ளன.

இந்த தொகுப்பை [நட்சத்திரம் செய்ய (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) மறக்காதீர்கள் மற்றும் [இந்த தொகுப்புக்கு fork செய்யவும்](https://github.com/microsoft/ai-agents-for-beginners/fork) குறியீட்டை இயக்க.

### தேவையானவை

இந்த பாடநெறியின் ஒவ்வொரு பாடத்திலும் குறியீட்டு உதாரணங்கள் உள்ளன, அவை code_samples கோப்புறையில் காணலாம். உங்கள் சொந்த நகலை உருவாக்க இந்த தொகுப்பை [fork செய்யலாம்](https://github.com/microsoft/ai-agents-for-beginners/fork).

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure கணக்கு தேவை

இந்த பாடநெறி Microsoft இன் பின்வரும் AI முகவர் கட்டமைப்புகளையும் சேவைகளையும் பயன்படுத்துகிறது:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
