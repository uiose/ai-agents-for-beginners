# AGENTS.md

## Project Overview

இந்த ரிப்பாசிட்டரி "ஆரம்பக்காரர்களுக்கான ஏஐ முகவர்கள்" - ஏஐ முகவர்களை உருவாக்க தேவையான அனைத்தையும் கற்பிக்கக்கூடிய விரிவான கல்விக் கொள்கை கொண்டுள்ளது. இந்த பாடநெறி அடிப்படைகள், வடிவமைப்பு மாதிரிகள், கட்டமைப்புக்கள் மற்றும் தயாரிப்பு விண்ணப்பம் ஆகிய 15+ பாடங்களை உள்ளடக்கியது.

**முக்கிய தொழில்நுட்பங்கள்:**
- Python 3.12+
- இணையவழி கற்றலுக்கு Jupyter நோட்புக்‌கள்
- ஏஐ கட்டமைப்புக்கள்: Microsoft Agent Framework (MAF)
- Azure AI சேவைகள்: Microsoft Foundry, Azure AI Foundry Agent Service V2

**வடிவமைப்பு:**
- பாடங்கள் அடிப்படையிலான அமைப்பு (00-15+ கோப்பகங்கள்)
- ஒவ்வொரு பாடமும்: README ஆவணங்கள், குறியீடு மாதிரிகள் (Jupyter நோட்புக்‌கள்), மற்றும் படங்கள் உள்ளன
- தானாக மொழிபெயர்க்கும் அமைப்பின் மூலம் பல மொழி ஆதரவு
- ஒவ்வொரு பாடத்திற்கும் Microsoft Agent Framework பயன்படுத்தும் ஒரு Python நோட்புக்

## Setup Commands

### தேவையானவை
- Python 3.12 அல்லது அதற்கு மேல்
- Azure சந்தா (Azure AI Foundryக்காக)
- Azure CLI நிறுவப்பட்டு அங்கீகாரம் பெற்றிருப்பது (`az login`)

### தொடக்க அமைப்பு

1. **ரிப்பாசிட்டரியை கிளோன் அல்லது ஃபோர்க் செய்யவும்:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # அல்லது
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python விர்சுவல் சுற்றுப்புறத்தை உருவாக்கி செயல்படுத்தவும்:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # விண்டோசில்: venv\Scripts\activate
   ```

3. **இணைப்புகளை நிறுவவும்:**
   ```bash
   pip install -r requirements.txt
   ```

4. **சுற்றுப்புற மாறிலிகளை அமைக்கவும்:**
   ```bash
   cp .env.example .env
   # உங்கள் API விசைகள் மற்றும் Endpoints உடன் .env ஐ திருத்துக
   ```

### தேவையான சுற்றுப்புற மாறிலிகள்

**Azure AI Foundry** (தேவையானவை):
- `AZURE_AI_PROJECT_ENDPOINT` - Azure AI Foundry திட்ட முடிவு இடம்
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - மாதிரி விண்ணப்பம் பெயர் (எ.கா., gpt-4o)

**Azure AI Search** (பாடம் 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search முடிவு இடம்
- `AZURE_SEARCH_API_KEY` - Azure AI Search API விசை

அங்கீகாரம்: நோட்புக்‌களை இயக்கும் முன் `az login` இயக்கவும் (`AzureCliCredential` பயன்படுத்துகிறது).
