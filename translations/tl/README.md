>
> Ang repositoryong ito ay may kasamang 50+ na pagsasalin ng wika na malaki ang dagdag sa laki ng pag-download. Para mag-clone na walang mga pagsasalin, gamitin ang sparse checkout:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Bibigyan ka nito ng lahat ng kailangan mo upang tapusin ang kurso ng mas mabilis ang pag-download.

## 🌱 Pagsisimula

Ang kursong ito ay may mga aralin na sumasaklaw sa mga pundasyon ng paggawa ng AI Agents. Ang bawat aralin ay may kanya-kanyang paksa kaya maaari kang magsimula kahit saan mo gusto!

May suporta sa maramihang wika para sa kursong ito. Puntahan ang aming [mga available na wika dito](../..).

Kung ito ang iyong unang pagkakataon na gumawa gamit ang mga Generative AI na modelo, tingnan ang aming [Generative AI For Beginners](https://aka.ms/genai-beginners) na kurso, na may 21 na mga aralin tungkol sa paggawa gamit ang GenAI.

Huwag kalimutang [i-star (🌟) ang repo na ito](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) at [i-fork ang repo na ito](https://github.com/microsoft/ai-agents-for-beginners/fork) upang patakbuhin ang code.

### Makipagkilala sa Ibang Nag-aaral, Makakuha ng Sagot sa Iyong mga Tanong

Kung ikaw ay maipit o may mga tanong tungkol sa paggawa ng AI Agents, sumali sa aming nakalaang Discord Channel sa [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Ano ang Kailangan Mo

Ang bawat aralin sa kursong ito ay may kasamang mga halimbawa ng code, na makikita sa code_samples na folder. Maaari mong [i-fork ang repo na ito](https://github.com/microsoft/ai-agents-for-beginners/fork) upang gumawa ng sarili mong kopya.

Ang mga halimbawa ng code sa mga ehersisyo ay gumagamit ng Microsoft Agent Framework kasama ang Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Kinakailangan ang Azure Account

Ang kursong ito ay gumagamit ng mga sumusunod na AI Agent frameworks at mga serbisyo mula sa Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Para sa dagdag na impormasyon sa pagpapatakbo ng code para sa kursong ito, puntahan ang [Course Setup](./00-course-setup/README.md).
