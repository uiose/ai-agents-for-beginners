> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git

> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Це дасть вам усе необхідне для проходження курсу з набагато швидшим завантаженням.

## 🌱 Початок роботи

Цей курс містить уроки, що охоплюють основи створення агентів ШІ. Кожен урок присвячений окремій темі, тож починайте звідки завгодно!

Для цього курсу доступна підтримка кількох мов. Перейдіть до наших [доступних мов тут](../..). 

Якщо це ваш перший досвід створення з моделями генеративного ШІ, ознайомтеся з нашим курсом [Генеративний ШІ для початківців](https://aka.ms/genai-beginners), який містить 21 урок щодо розробки з GenAI.

Не забудьте [поставте (🌟) зірку цьому репозиторію](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) та [форкніть цей репозиторій](https://github.com/microsoft/ai-agents-for-beginners/fork), щоб запустити код.

### Познайомтеся з іншими учнями та отримуйте відповіді на свої запитання

Якщо ви застрягли або маєте запитання щодо створення агентів ШІ, приєднуйтеся до нашого спеціального каналу Discord у [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Що вам потрібно 

Кожен урок у цьому курсі містить приклади коду, які можна знайти в папці code_samples. Ви можете [форкнути цей репозиторій](https://github.com/microsoft/ai-agents-for-beginners/fork), щоб створити свою копію.  

Приклади коду в цих вправах використовують Microsoft Agent Framework з Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Потребується обліковий запис Azure

У цьому курсі використовуються такі фреймворки та сервіси для агентів ШІ від Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Щоб дізнатися більше про запуск коду для цього курсу, перейдіть до [Налаштування курсу](./00-course-setup/README.md).
