# Пример за Github MCP сървър

## Описание

Това беше демо, създадено за AI Agents Hackathon, провеждан от Microsoft Reactor.

Инструментът се използва за препоръчване на проекти за хакатон въз основа на Github репозиториите на потребителя.
Това се прави чрез:

1. **Github Agent** - Използване на Github MCP Server за извличане на репозитории и информация за тези репозитории.
2. **Hackathon Agent** - Взема данните от Github Agent и генерира творчески идеи за проекти за хакатон въз основа на проектите, езиците, използвани от потребителя, и тематичните направления на AI Agents Hackathon.
3. **Events Agent** - Въз основа на предложенията на Hackathon Agent, Events Agent ще препоръча релевантни събития от серията AI Agent Hackathon.
## Стартиране на кода 

### Променливи на средата

Това демо използва Microsoft Agent Framework, Azure OpenAI Service, Github MCP Server и Azure AI Search.

Уверете се, че имате зададени правилните променливи на средата, за да използвате тези инструменти:

```python
AZURE_AI_PROJECT_ENDPOINT=""
AZURE_AI_MODEL_DEPLOYMENT_NAME=""
AZURE_SEARCH_SERVICE_ENDPOINT=""
AZURE_SEARCH_API_KEY=""
``` 

## Стартиране на Chainlit сървъра

За да се свържете с MCP сървъра, това демо използва Chainlit като чат интерфейс. 

За да стартирате сървъра, използвайте следната команда в терминала си:

```bash
chainlit run app.py -w
```

Това трябва да стартира Chainlit сървъра ви на `localhost:8000`, както и да попълни вашия Azure AI Search Index със съдържанието на `event-descriptions.md`. 

## Свързване с MCP сървър

За да се свържете с Github MCP Server, изберете иконата "plug" под полето за чат "Type your message here..":

![Свързване с MCP](../../../../../translated_images/bg/mcp-chainlit-1.7ed66d648e3cfb28.webp)

Оттам можете да кликнете върху "Connect an MCP" за да добавите командата за свързване към Github MCP Server:

```bash
npx -y @modelcontextprotocol/server-github --env GITHUB_PERSONAL_ACCESS_TOKEN=[YOUR PERSONAL ACCESS TOKEN]
```

Заменете "[YOUR PERSONAL ACCESS TOKEN]" с вашия реален персонален токен за достъп. 

След свързване трябва да видите (1) до иконата plug, за да потвърдите, че е свързано. Ако не, опитайте да рестартирате chainlit сървъра с `chainlit run app.py -w`.

## Използване на демото 

За да стартирате работния процес на агентите за препоръчване на проекти за хакатон, можете да напишете съобщение като: 

"Препоръчайте проекти за хакатон за Github потребителя koreyspace"

Router Agent ще анализира вашата заявка и ще определи коя комбинация от агенти (GitHub, Hackathon и Events) е най-подходяща за обработка на вашето запитване. Агентите работят заедно, за да предоставят цялостни препоръки, базирани на анализ на GitHub репозитории, генериране на идеи за проекти и релевантни технологични събития.
