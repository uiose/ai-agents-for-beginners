# AGENTS.md

## Преглед на проекта

Този репозиторий съдържа "AI Агенти за начинаещи" - цялостен образователен курс, който учи всичко необходимо за изграждане на AI агенти. Курсът се състои от над 15 урока, обхващащи основите, дизайнерски модели, рамки и продукционно внедряване на AI агенти.

**Основни технологии:**
- Python 3.12+
- Jupyter тетрадки за интерактивно обучение
- AI рамки: Microsoft Agent Framework (MAF)
- Azure AI услуги: Microsoft Foundry, Azure AI Foundry Agent Service V2

**Архитектура:**
- Структура на базата на уроци (директории 00-15+)
- Всеки урок съдържа: README документация, кодови примери (Jupyter тетрадки) и изображения
- Поддръжка на много езици чрез автоматична система за превод
- Една Python тетрадка на урок с използване на Microsoft Agent Framework

## Команди за настройка

### Предварителни изисквания
- Python 3.12 или по-висока версия
- Абонамент за Azure (за Azure AI Foundry)
- Инсталиран и автентикиран Azure CLI (`az login`)

### Първоначална настройка

1. **Клонирайте или форкнете репозитория:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # ИЛИ
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Създайте и активирайте виртуална Python среда:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # В Windows: venv\Scripts\activate
   ```

3. **Инсталирайте зависимостите:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте променливи на средата:**
   ```bash
   cp .env.example .env
   # Редактирайте .env с вашите API ключове и крайни точки
   ```

### Задължителни променливи на средата

За **Azure AI Foundry** (задължително):
- `AZURE_AI_PROJECT_ENDPOINT` - крайна точка на Azure AI Foundry проект
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - име на разгръщане на модел (напр. gpt-4o)

За **Azure AI Search** (Урок 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - крайна точка на Azure AI Search
- `AZURE_SEARCH_API_KEY` - API ключ за Azure AI Search

Автентикация: Стартирайте `az login` преди изпълнение на тетрадките (използва `AzureCliCredential`).

## Работен процес за разработка

### Стартиране на Jupyter тетрадки

Всеки урок съдържа няколко Jupyter тетрадки за различни рамки:

1. **Стартирайте Jupyter:**
   ```bash
   jupyter notebook
   ```

2. **Навигирайте до директорията на урока** (напр. `01-intro-to-ai-agents/code_samples/`)

3. **Отворете и изпълнете тетрадките:**
   - `*-python-agent-framework.ipynb` - Използване на Microsoft Agent Framework (Python)
   - `*-dotnet-agent-framework.ipynb` - Използване на Microsoft Agent Framework (.NET)

### Работа с Microsoft Agent Framework

**Microsoft Agent Framework + Azure AI Foundry:**
- Изисква абонамент за Azure
- Използва `AzureAIProjectAgentProvider` за Agent Service V2 (агентите се виждат в портала Foundry)
- Готов за продукция с вградена наблюдаемост
- Патерн на файла: `*-python-agent-framework.ipynb`

## Контекст, специфичен за проекта

### Поддръжка на много езици

Този репозиторий използва автоматична система за превод:
- Поддържа 50+ езика
- Преводите са в директории `/translations/<lang-code>/`
- GitHub Actions workflow управлява обновленията на преводите
- Изходните файлове са на английски в корена на репозитория

### Структура на уроците

Всеки урок следва последователен модел:
1. Видео миниатюра с линк
2. Писмено съдържание на урока (README.md)
3. Кодови примери в различни рамки
4. Образователни цели и предварителни изисквания
5. Допълнителни ресурси за обучение

### Именуване на кодови примери

Формат: `<lesson-number>-python-agent-framework.ipynb`
- `01-python-agent-framework.ipynb` - Урок 1, MAF Python
- `14-sequential.ipynb` - Урок 14, напреднали MAF модели

### Специални директории

- `translated_images/` - Локализирани изображения за преводи
- `images/` - Оригинални изображения за съдържание на английски
- `.devcontainer/` - Конфигурация на контейнер за разработка VS Code
- `.github/` - GitHub Actions workflows и шаблони

### Зависимости

Основни пакети от `requirements.txt`:
- `agent-framework` - Microsoft Agent Framework
- `a2a-sdk` - Поддръжка на Agent-to-Agent протокол
- `azure-ai-inference`, `azure-ai-projects` - Azure AI услуги
- `azure-identity` - Azure автентикация (AzureCliCredential)
- `azure-search-documents` - Интеграция с Azure AI Search
- `mcp[cli]` - Поддръжка на Model Context Protocol
