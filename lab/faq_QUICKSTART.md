# 快速开始：5分钟运行你的第一个 Agent

## 第一步：安装依赖（2分钟）

```bash
cd d:\agentstart\ai-agents-for-beginners\lab

# 确保虚拟环境激活
..\venv\Scripts\activate

# 安装所需的包
pip install openai python-dotenv
```

## 第二步：配置 API 密钥（1分钟）

1. 打开 `.env.example` 文件
2. 复制内容，在同目录新建 `.env` 文件
3. 填入你的 OpenAI API 密钥：
   ```
   OPENAI_API_KEY=sk-xxxxxxxxxxxxxx
   ```
4. 保存

## 第三步：运行项目（2分钟）

```bash
python faq_agent_starter.py
```

你会看到：
