# Demo Questions

这份清单可以直接拿来做今晚的测试和明天的组会展示。

## 建议先跑的 5 个问题

1. 什么是 agentic framework？
2. Microsoft Agent Framework 和 Azure AI Agent Service 有什么区别？
3. 什么是工具使用设计模式？
4. 在 Azure AI Foundry 中创建 agent 时需要关注哪些关键元素？
5. 为什么做课程资料问答时要用检索增强，而不是只靠大模型直接回答？

## 建议保留的结果

- 2 个效果比较好的案例
- 1 个一般或失败的案例
- 每个案例保留：
  - 用户问题
  - 检索到的上下文
  - Agent 最终回答
  - 你自己的一句分析

## 失败案例怎么讲

如果结果不理想，不要怕，这反而是组会上很正常也很有价值的部分。你可以这样总结：

- 当前原型已经验证了“检索 + 回答”的主链路是可行的。
- 但本地轻量检索还比较粗糙，相关段落排序有时不稳定。
- 下一步准备换成更标准的检索方案，并补充评测集。
